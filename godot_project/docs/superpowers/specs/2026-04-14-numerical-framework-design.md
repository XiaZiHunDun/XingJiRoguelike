```
【基础攻击公式】
├─ 物理攻击 = 体质×2 + 武器基础攻击
└─ 魔法攻击 = 精神×2 + 武器基础攻击

├─ 巨剑（体质型）：体质×1.5 + 10 = 290

├─ 巨剑：187×2 + 290 = 664

└─ 双刃：142×2 + 179 = 463
```

### 1.4 防御属性

```
【基础防御公式】
├─ 物理防御 = 体质×2 + 装备防御
└─ 魔法抗性 = 精神×2 + 装备魔抗

【50级基础防御（无装备）】
├─ 体质防御：187×2 = 374
└─ 精神魔抗：142×2 = 284

【防御减伤公式】
减伤比例 = 防御 / (防御 + 100)

【防御减伤参考】
├─ 100防御 → 50%减伤

├─ 300防御 → 75%减伤

└─ 500防御 → 83%减伤
```

## 3. 战斗伤害计算

### 3.1 伤害公式

```
【物理伤害】
原始伤害 = 物理攻击 × 技能倍率 × 时机加成
最终伤害 = 原始伤害 × (1 - 伤害减免) - 防御力

【魔法伤害】
原始伤害 = 魔法攻击 × 技能倍率 × 时机加成
最终伤害 = 原始伤害 × (1 - 伤害减免) - 魔抗

【时机加成】
├─ 完美时机(100%)：×1.15
├─ 良好时机(90-99%)：×1.05
├─ 普通时机(70-89%)：×1.0
└─ 仓促时机(<70%)：×0.8
```

### 3.2 50级玩家伤害计算示例

```
【条件】
├─ 玩家：50级，巨剑
├─ 物理攻击：664（无装备）
├─ 技能：横斩（100%攻击，15% ATB消耗）
├─ 目标：50级普通怪
└─ 防御：100

【计算】
├─ 原始伤害 = 664 × 1.0 × 1.0 = 664
├─ 伤害减免 = 100/(100+100) = 50%
├─ 最终伤害 = 664 × (1-0.5) - 100 = 332 - 100 = 232
└─ 击杀需要：1050 / 232 ≈ 5下
```

### 3.3 装备加成后伤害计算

```
【条件】
├─ 玩家：50级，巨剑
├─ 物理攻击：664 + 装备200 = 864
├─ 词缀加成：物理伤害+20%
├─ 技能：横斩（100%攻击）
├─ 目标：50级普通怪（防御100）
└─ 时机：普通

【计算】
├─ 最终攻击 = 864 × 1.2 = 1036.8
├─ 原始伤害 = 1036.8 × 1.0 × 1.0 = 1036.8
├─ 伤害减免 = 50%
├─ 最终伤害 = 1036.8 × 0.5 - 100 = 518.4 - 100 = 418.4
└─ 击杀需要：1050 / 418 ≈ 3下
```

### 3.4 BOSS战伤害计算

```
【条件】
├─ 玩家：50级，巨剑，装备加成后攻击1200
├─ 技能：流星（200%攻击，50% ATB消耗）
├─ 目标：50级BOSS（防御300，魔抗300）
└─ 伤害减免：300/(300+100) = 75%

【计算】
├─ 原始伤害 = 1200 × 2.0 = 2400
├─ 防御减伤 = 75%
├─ 最终伤害 = 2400 × 0.25 - 300 = 600 - 300 = 300
└─ BOSS HP 21000，需要70下（理论值）
```

## 4. 战斗节奏

### 4.1 击杀时间参考

```
【单挑普通怪】
├─ 低级（1-10级）：2-3下，约5-10秒
├─ 中级（15-25级）：3-5下，约10-15秒
├─ 高级（30-40级）：4-6下，约15-20秒
└─ 满级（50级）：3-5下，约10-15秒

【单挑精英怪】
├─ 需要10-20下
├─ 战斗时间：30-60秒
└─ 需要使用技能和道具

【BOSS战】
├─ 需要50-100下
├─ 战斗时间：2-5分钟
└─ 需要合理使用技能和ATB管理
```

### 4.2 ATB速度与行动次数

```
【50级玩家ATB速度】
基础速度 = 100 + 142×3 = 526
（溢出326点，伤害+326%）

【蓄力时间】
├─ 满ATB槽 = 100% / (526/10) = 1.9秒
└─ 约每2秒可行动一次

【一场战斗（10下击杀）】
├─ 约20秒
├─ 可行动10次左右
└─ 合理分配普攻和技能
```

## 7. 特殊数值机制

### 7.1 速度溢出机制

```
【ATB速度上限】200（软上限）

【溢出计算】
溢出伤害加成 = (ATB速度 - 200) × 1%

【示例】
├─ ATB速度250：伤害+50%
├─ ATB速度300：伤害+100%
├─ ATB速度400：伤害+200%
└─ ATB速度527（50级无装备）：伤害+327%

【设计目的】
├─ 高速度不仅是快速行动
├─ 还能获得额外伤害加成
└─ 鼓励速度Build
```

### 7.2 闪避与命中

```
【闪避率公式】
闪避率 = min(50%, 敏捷×0.4% + 装备加成)

【50级闪避率】
├─ 敏捷142：闪避率 = min(50%, 142×0.4%) = min(50%, 56.8%) = 56.8%
└─ 实际取50%上限

【命中公式】
命中基础值 = 100%
BOSS最低命中率 = 20%
```

### 7.3 暴击机制

```
【暴击率公式】
暴击率 = min(80%, 敏捷×0.5% + 装备加成)

【暴击伤害公式】
暴击伤害 = 150% × (1 + 敏捷×0.2% + 装备加成%)

【50级暴击】
├─ 敏捷142：暴击率 = min(80%, 71%) = 71%
└─ 暴击伤害 = 150% × (1 + 28.4%) = 192.6%
```

## 8. 数值设计原则

### 8.1 设计目标

```
【战斗时长目标】
├─ 普通怪：5-15秒
├─ 精英怪：30-60秒
└─ BOSS：2-5分钟

【玩家成长感】
├─ 每10级属性翻倍
├─ 装备提供50-100%的属性加成
└─ 词缀提供20-50%的伤害提升

【Build多样性】
├─ 物理Build：追求高攻击
├─ 速度Build：追求ATB溢出伤害
├─ 暴击Build：追求高暴击率
└─ 防御Build：追求高生存
```

### 8.2 调整方向

```
【如果战斗太简单】
├─ 提高敌人攻击力
├─ 降低敌人HP
└─ 增加敌人防御

【如果战斗太难】
├─ 提高玩家攻击力
├─ 降低敌人攻击力
├─ 提高玩家HP
└─ 添加更多治疗手段

【如果战斗太短/太长】
├─ 调整ATB速度
├─ 调整敌人HP
└─ 调整技能伤害倍率
```
