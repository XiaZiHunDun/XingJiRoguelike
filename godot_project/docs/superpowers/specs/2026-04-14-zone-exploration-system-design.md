## 2. 区域内部结构

### 2.1 线性推进机制

```
【区域结构示例：沙海回声（起始等级1）】
┌─────────────────────────────────────────────────────────┐
│                     据点（传送门）                         │
│                   [安全区/商店/任务]                       │
└─────────────────────────────────────────────────────────┘
                         ↓
              ┌─────────────────┐
              │     地图1       │ ← 新手练级区
              │   1-14级怪物    │
              └─────────────────┘
                         ↓ (击败BOSS后解锁)
              ┌─────────────────┐
              │     地图2       │ ← 中级练级区
              │   1-14级怪物    │
              └─────────────────┘
                         ↓ (击败BOSS后解锁)
              ┌─────────────────┐
              │     地图3       │ ← 中高级练级区
              │   15-28级怪物   │
              └─────────────────┘
                         ↓ (击败BOSS后解锁)
              ┌─────────────────┐
              │     地图4       │ ← 高级练级区
              │   15-28级怪物   │
              └─────────────────┘
                         ↓ (击败BOSS后解锁)
              ┌─────────────────┐
              │     地图5       │ ← 区域BOSS（满级）
              │   57-70级怪物   │
              └─────────────────┘

【其他区域结构相同，地图等级按起始等级递增】
霜棘王庭（起始10级）：地图1-2为10-23级，地图3-4为24-37级，地图5为57-70级
翠蔓圣所（起始15级）：地图1-2为15-28级，地图3-4为29-42级，地图5为57-70级
机魂废土（起始25级）：地图1-2为25-38级，地图3-4为39-52级，地图5为57-70级
太初核心（起始50级）：地图1-2为50-63级，地图3-4为50-63级，地图5为57-70级

【推进规则】
├─ 必须按顺序通关（1→2→3→4→5）
├─ 击败地图5的BOSS后，传送门激活
├─ 地图1-4可重复进入（练级/刷装备）
└─ 地图5通关后，该区域完成，可前往下个区域
```

### 2.2 地图类型

```
【普通地图】
├─ 包含：战斗事件、采集点、随机事件
├─ 敌人：3-5波普通怪 + 1波精英
├─ 地形：根据区域文化变化
└─ 通关：消灭所有敌人

【BOSS地图】
├─ 只包含：BOSS战
├─ BOSS：区域之主
├─ 掉落：套装组件、记忆碎片、永久增幅物品
└─ 通关后：激活传送门，解锁下一个区域入口

【特殊地图】
├─ 商店地图：进入后显示商店界面
├─ 事件地图：随机事件（有利/不利）
└─ 休息地图：完全恢复HP/能量
```

### 2.3 地图内结构

```
【单张地图布局】
┌─────────────────────────────────────────────────────────┐
│  [入口]                                                   │
│     ↓                                                     │
│  ┌───┐  ┌───┐  ┌───┐                                    │
│  │ 1 │──│ 2 │──│ 3 │  ← 战斗区域（随机生成敌人位置）       │
│  └───┘  └───┘  └───┘                                    │
│     │      │      │                                       │
│  ┌───┐  ┌───┐  ┌───┐                                    │
│  │ 4 │──│ 5 │──│ 6 │  ← 更多战斗区域                     │
│  └───┘  └───┘  └───┘                                    │
│                                                         │
│                      [采集点] [随机事件]                   │
│                                                         │
│                         ↓                                │
│                   [BOSS区域/出口]                          │
└─────────────────────────────────────────────────────────┘

【地图规则】
├─ 每张地图包含6-10个房间/区域
├─ 敌人位置随机生成
├─ 采集点、事件点随机分布
├─ BOSS房间在地图最深处
└─ 通关后返回据点
```

## 4. 传送门系统

### 4.1 传送门激活

```
【激活条件】
├─ 击败区域BOSS（地图5）
├─ 激活后显示金色光效
└─ 全局保留（死亡后不消失）

【传送门位置】
├─ 每个区域1个传送门（位于据点）
├─ 连接据点与区域入口
└─ 区域间传送需要通过据点
```

### 4.2 传送功能

```
【据点传送】
┌─────────────────────────────────────────────────────────┐
│ 据点A ←→ 区域A ←→ 据点B ←→ 区域B ←→ 据点C...          │
└─────────────────────────────────────────────────────────┘

【传送流程】
1. 进入据点
2. 选择地图（地图1-5，已解锁的）
3. 点击进入
4. 直接传送到对应地图入口
```

## 5. 探索流程

### 5.1 单次探索流程

```
【完整探索流程】

1. 选择区域
   └─ 从据点选择要进入的区域

2. 选择地图
   └─ 选择地图1-5中的某一个（已解锁）

3. 进入地图
   └─ 加载地图，初始化敌人/事件

4. 战斗/探索
   ├─ 击败敌人获得装备/星尘
   ├─ 采集材料
   └─ 触发随机事件

5. 通关判定
   ├─ 击败所有敌人 → 通关
   ├─ 击败BOSS → 激活传送门
   └─ 死亡 → 回到据点，损失20%星尘

6. 返回据点
   └─ 通关后返回据点，可继续探索/强化/出售
```

### 5.2 死亡处理

```
【死亡规则】
├─ HP归0 → 战斗失败
├─ 损失：当前星尘的20%
├─ 损失：未拾取的掉落物
├─ 保留：已装备的物品
├─ 保留：已激活的传送门
├─ 保留：已完成的任务进度
└─ 保留：永久增幅物品

【死亡后】
├─ 返回最近据点
├─ HP/能量完全恢复
└─ 可立即继续游戏
```

## 6. 随机事件

### 6.1 事件类型

```
【有利事件】
├─ 发现宝箱：获得装备/星尘
├─ 商人出现：可购买物品
├─ 休息点：完全恢复HP/能量
├─ 增益神龛：临时属性提升
└─ 剧情片段：了解世界观

【中立事件】
├─ 随机遭遇：精英敌人
├─ 陷阱触发：损失HP/获得负面状态
└─ 谜题挑战：答对获得奖励，答错受惩罚

【不利事件】
├─ 敌人埋伏：强制战斗
├─ 地形陷阱：持续伤害
└─ 诅咒神龛：临时属性下降
```

### 6.2 事件触发

```
【触发机制】
├─ 每张地图有10-20%概率生成随机事件
├─ 事件在进入地图时判定
├─ 部分事件可选择是否触发
└─ BOSS地图不触发随机事件
```

## 7. 地图生成

### 7.1 地图元素

```
【地图组成】
├─ 房间：6-10个房间/区域
├─ 通道：连接房间的路径
├─ 敌人：3-5波普通怪 + 1波精英
├─ 采集点：1-3个（矿石/药材）
├─ 事件点：0-1个（随机）
└─ BOSS房：1个（仅地图5）

【地形元素】
├─ 障碍物：阻挡移动/技能
├─ 元素池：环境效果（如有）
├─ 可破坏物：摧毁后获得物品
└─ 隐藏区域：探索发现额外奖励
```

### 7.2 敌人配置

```
【普通地图敌人配置】
├─ 怪物波次：3-5波
├─ 每波怪物：2-4只
├─ 精英出现：每张地图1-2次
└─ 敌人等级：地图1-2为1-14级，地图3-4为15-28级

【BOSS地图敌人配置】
├─ BOSS：1只（区域之主，57-70级）
├─ 小怪：0-2波（57-70级辅助BOSS）
└─ 不可逃跑
```

## 8. 区域难度设计

### 8.1 难度梯度

| 区域 | 起始等级 | 地图1-2 | 地图3-4 | 地图5 |
|------|----------|---------|---------|-------|
| 沙海回声 | 1级 | 1-14级 | 15-28级 | 57-70级 |
| 霜棘王庭 | 10级 | 10-23级 | 24-37级 | 57-70级 |
| 翠蔓圣所 | 15级 | 15-28级 | 29-42级 | 57-70级 |
| 机魂废土 | 25级 | 25-38级 | 39-52级 | 57-70级 |
| 太初核心 | 50级 | 50-63级 | 50-63级 | 57-70级 |

**说明：** 所有区域的地图5都是57-70级满级区域，玩家可以自由选择在任何区域的地图5进行高级练级。

### 8.2 敌人强度调整

```
【敌人强度】
├─ HP：随等级提升
├─ 伤害：随等级提升
├─ 防御：随等级提升
└─ 技能：更高等级解锁更多技能

【玩家适配】
├─ 玩家境界越高，敌人等级越高
├─ 区域锁定最低等级（避免玩家越级挑战）
└─ 装备等级由怪物等级决定
```
