## 1. 核心设计

### 1.1 设计原则

```
【取消】
├─ 稀有度系统（凡品/稀有/史诗/传说）
├─ 装等系统（T1-T6）
└─ 装备分解系统

【替代方案】
├─ 穿戴要求系统：代替装等/稀有度作为装备门槛
├─ 只能出售：装备作为资源而非燃料
└─ 技能+词缀随机：每件装备独特
```

### 1.2 装备掉落机制

```
【技能数量分布】（各20%均匀分布）
├─ 0技能：20%（纯词缀装）
├─ 1技能：20%
├─ 2技能：20%
├─ 3技能：20%
└─ 4技能：20%

【词缀数量】
├─ 范围：1-4个词缀
└─ 随机：独立随机

【穿戴要求】（三维随机）
├─ 要求数量：0-3个（随机）
├─ 要求类型：属性/境界/技能（随机选择）
└─ 详见第二章
```

## 2. 穿戴要求系统

### 2.1 要求类型

| 要求类型 | 示例 | 效果 |
|----------|------|------|
| 属性要求 | 体质≥20，精神≥15 | 属性达标才能穿戴 |
| 境界要求 | 凡人境10级 | 境界等级达标才能穿戴 |
| 技能要求 | 横斩Lv2，疾跑Lv1 | 特定技能等级达标才能穿戴 |

### 2.2 要求生成规则

```
【生成流程】
1. 随机决定要求数量（0-3个）
2. 每种要求独立随机
3. 同一类型要求可能出现多个

【示例】
装备A：仅有属性要求（体质≥25）→ 玩家体质25可穿戴
装备B：属性+境界（体质≥20，凝核境）→ 两者都满足才可穿
装备C：技能+境界（横斩Lv2，凡人境20级）→ 两者都满足才可穿
装备D：三重要求（体质≥30，精神≥20，技能横斩Lv1）→ 全部满足才可穿
装备E：无要求（白板装）→ 可直接穿戴
装备F：玩家不满足要求 → 不可穿戴，但可正常出售
```

### 2.3 穿戴失败处理

```
【穿戴规则】
├─ 属性/境界/技能要求全部满足 → 可穿戴
├─ 任意一项不满足 → 完全无法穿戴
└─ 不可穿戴的装备可正常出售

【示例】
装备A：体质≥25要求 → 玩家体质25 → 可穿戴
装备B：体质≥30要求 → 玩家体质25 → 不可穿戴，可出售
装备C：体质≥30+横斩Lv2 → 玩家体质25，技能满足 → 不可穿戴
```

## 4. 装备数值

### 4.1 武器基础攻击

```
【武器基础攻击公式】
基础攻击 = 主属性系数 × 主属性 + 固定值

【示例（50级角色）】
巨剑（体质型）：体质130 → 基础攻击 = 130×1.5+10 = 205
法杖（精神型）：精神105 → 基础攻击 = 105×1.5+10 = 167.5
双刃（敏捷型）：敏捷90 → 基础攻击 = 90×1.2+8 = 116
```

### 4.2 护甲基础防御

```
【护甲基础防御公式】
钢甲防御 = 体质 × 2 + 15
法袍魔抗 = 精神 × 2 + 15
皮甲防御 = 敏捷 × 1.5 + 8
修真袍防御 = (体质+精神) × 1.5 + 10

【示例（50级角色）】
钢甲（体质130）：防御 = 130×2+15 = 275
法袍（精神105）：魔抗 = 105×2+15 = 225
皮甲（敏捷90）：防御 = 90×1.5+8 = 143
修真袍（体质130+精神105）：防御 = (130+105)×1.5+10 = 362.5
```

## 5. 唯一性装备

```
【世界唯一装备】
├─ 同一时间整个世界只有一件
├─ 死亡后消失，可再次获取
├─ 获取方式：
│   ├─ 任务奖励
│   ├─ 成就奖励

│   └─ 极低概率随机掉落（<1%）

## 6. 装备出售与经济

### 6.1 装备定价

```
【装备售价公式】
售价 = 基础价格 × (1 + 技能数量 × 0.2) × (1 + 词缀数量 × 0.1)

【基础价格】
├─ 武器：30星尘
├─ 护甲：25星尘
├─ 饰品：20星尘
└─ 增幅器：不可出售

【示例】
0技能+1词缀武器：30 × 1.0 × 1.1 = 33星尘
2技能+3词缀武器：30 × 1.4 × 1.3 = 54.6 ≈ 55星尘
4技能+4词缀武器：30 × 1.8 × 1.4 = 75.6 ≈ 76星尘
```

### 6.2 商店系统

```
【商店功能】
├─ 购买装备（随机刷新）
├─ 购买消耗品
├─ 购买材料
└─ 出售装备

【商店刷新】
├─ 每次进入据点刷新一次
├─ 手动刷新消耗星尘
└─ 刷新价格：10星尘/次
```

## 7. 重铸系统

### 7.1 重铸类型

```
【词缀重铸】
├─ 消耗：10星尘
├─ 效果：随机改变1个词缀
└─ 限制：已有4个词缀的装备不能再重铸

【技能重铸】
├─ 消耗：20星尘
├─ 效果：随机改变技能组合
└─ 限制：0技能装备不能重铸技能

【解除穿戴要求】
├─ 消耗：15星尘
├─ 效果：随机移除1个穿戴要求
└─ 限制：无要求的装备不能解除
```

### 7.2 重铸限制

```
【重铸规则】
├─ 每次重铸随机生成新结果
├─ 重铸可能变好也可能变差
├─ 这是Roguelike的随机性体现
└─ 玩家需要权衡是否值得重铸
```

## 8. 装备管理

### 8.1 背包系统

```
【背包容量】
├─ 初始容量：50格
├─ 扩展方式：消耗道具扩展

```
【对比系统】
├─ 显示当前装备 vs 掉落装备

```
