## 1. 系统概述

```
【系统定位】
├─ 局内经济循环的重要组成部分
├─ 用于制作消耗品、强化装备
└─ 采集物是Roguelike探索的奖励之一

【物资分类】
├─ 矿石类：用于装备制作和强化
├─ 药材类：用于制作消耗品
├─ 特殊材料：各区域特色材料
└─ 异兽核心：高级制作材料
```

## 2. 采集点机制

### 2.1 采集点刷新

```
【刷新规则】
├─ 每次进入地图时重新生成采集点
├─ 采集后消失，该局内无法再次采集
└─ 下一局进入时重新生成

【采集点数量】
├─ 每张地图：1-3个采集点
├─ 地图难度越高，采集点可能越少（但质量更高）
└─ BOSS地图通常没有采集点
```

### 2.2 采集操作

```
【采集流程】
1. 靠近采集点
2. 显示采集提示
3. 点击采集按钮
4. 消耗一定时间（1-2秒）
5. 获得随机物资

【采集限制】
├─ 无次数限制（有多少采多少）
└─ 采集点采集后消失
```

## 3. 矿石类物资

### 3.1 矿石种类

| 矿石名称 | 等级 | 用途 | 分布 |
|----------|------|------|------|
| 铁矿石 | 1级 | 基础合成 | 所有区域 |
| 精炼锭 | 1级 | 装备强化 | 所有区域（需合成） |

| 陨星碎片 | 50级 | 唯一性装备制作 | 57-70级区域 |

### 3.2 矿石获取方式

```
【获取方式】
├─ 地图采集：主要来源
├─ 怪物掉落：普通怪/精英/BOSS
└─ 商店购买：基础矿石可购买

【采集概率】
铁矿石：40%
精炼锭：25%（需要合成）
星银矿：15%（29级以上区域）
陨星碎片：5%（57-70级区域）
```

【获取方式】

## 6. 消耗品制作

### 6.1 制作场所

```
【制作场所】
├─ 据点工坊：完全制作能力
├─ 便携制作台：消耗星尘，可在局内使用
└─ 商店购买：基础消耗品

【制作费用】
├─ 据点工坊：免费
└─ 便携制作台：消耗星尘（5星尘/次）
```

### 6.2 制作配方

```
【基础消耗品】
├─ 初级HP药水：止血草×2 → 恢复30%HP
├─ 初级能量药水：灵力花×2 → 恢复1点能量
├─ 解毒剂：解毒蕨×1 → 解除一个负面状态
└─ 初级体力药剂：疾风藤×2 → ATB速度+30%，3回合

【中级消耗品】
├─ 中级HP药水：止血草×3 + 异兽核心×1 → 恢复50%HP
├─ 中级能量药水：灵力花×3 + 异兽核心×1 → 恢复2点能量
├─ 护盾药剂：护盾苔×2 + 异兽核心×1 → 获得100点护盾
└─ 抗性药剂：区域特色药材×2 → 对应抗性+30%

【高级消耗品】（需要特定材料）
├─ 高级HP药水：止血草×5 + 异兽核心×2 → 恢复100%HP
├─ 高级能量药水：灵力花×5 + 异兽核心×2 → 恢复全部能量
├─ 虚弱药剂：区域特色药材×3 → 使敌人攻击-20%，持续30秒
└─ 力量药剂：区域特色药材×3 → 使我方攻击+20%，持续30秒
```

### 6.3 配方解锁

```
【解锁方式】
├─ 境界提升：解锁更多配方
├─ 任务奖励：特殊配方
└─ 探索发现：隐藏配方

【境界与配方】
├─ 凡人身（1-10级）：初级消耗品
├─ 感应境（11-20级）：中级消耗品
├─ 聚尘境（21-30级）：部分高级消耗品
├─ 凝核境（31-40级）：大部分高级消耗品
└─ 星火境（41-50级）：全部配方
```

## 7. 物资管理

### 7.1 物品堆叠

```
【堆叠规则】
├─ 矿石：99个堆叠
├─ 药材：99个堆叠
├─ 消耗品：99个堆叠
├─ 特殊材料：99个堆叠
└─ 异兽核心：10个堆叠（稀有）
```

### 7.2 物品交易

```
【交易规则】
├─ 矿石：可出售给商店
├─ 药材：可出售给商店
├─ 消耗品：可出售给商店
├─ 特殊材料：不可出售
└─ 异兽核心：不可出售

【出售价格】
├─ 铁矿石：1星尘
├─ 星银矿：5星尘
├─ 陨星碎片：20星尘
├─ 止血草：2星尘
├─ 灵力花：3星尘
└─ 异兽核心：不可出售
```

## 9. 采集点类型

### 9.1 采集点外观

```
【矿石采集点】
├─ 外观：岩石堆、矿脉
├─ 颜色：灰色/金属色
└─ 采集物：铁矿石、星银矿、陨星碎片

【植物采集点】
├─ 外观：灌木丛、花朵
├─ 颜色：绿色/彩色
└─ 采集物：止血草、灵力花、区域特色药材

【特殊采集点】
├─ 外观：发光符文、古老祭坛
├─ 颜色：金色/紫色
└─ 采集物：星尘粉、永久增幅物品
```

### 9.2 采集点分布

```
【地图内采集点分布】
├─ 每张地图1-3个采集点
├─ 采集点位置随机
├─ 采集点有明确视觉标识
└─ 靠近时显示采集提示

【采集点刷新】
└─ 每次进入地图时重新生成
```
