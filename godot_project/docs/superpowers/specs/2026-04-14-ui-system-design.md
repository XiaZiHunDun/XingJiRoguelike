## 1. UI设计规范

### 1.1 设计风格

```
【科幻/赛博风格】
├─ 主色调：深蓝黑色背景
├─ 强调色：霓虹蓝（#00D4FF）、霓虹紫（#A855F7）
├─ 警示色：霓虹红（#FF3366）
├─ 成功色：霓虹绿（#00FF88）
├─ 文字色：白色/浅灰色
└─ 边框效果：发光描边、扫描线效果
```

### 1.2 色彩系统

```
【主色板】
├─ 背景色：#0A0E14（深蓝黑）
├─ 面板背景：#1A1F2E（暗蓝灰）
├─ 边框色：#2A3040（灰蓝）
├─ 霓虹蓝：#00D4FF（主要强调色）
├─ 霓虹紫：#A855F7（次要强调色）
├─ 霓虹红：#FF3366（危险/警告）
├─ 霓虹绿：#00FF88（成功/治疗）
├─ 霓虹黄：#FFD700（重要/高亮）
├─ 文字白：#E8E8E8
└─ 文字灰：#888888
```

### 1.3 字体系统

```
【字体层级】
├─ 标题字体：Bold，24-32px
├─ 副标题：SemiBold，18-20px
├─ 正文：Regular，14-16px
├─ 辅助文字：Regular，12px
├─ 数值显示：Mono/Bold，16-24px
└─ 伤害数字：Bold，20-36px（根据伤害变化）
```

└─ 跟随目标位置飘出，逐渐消失

## 8. 快捷键系统

### 8.1 战斗快捷键

```
【技能快捷键】
├─ 1-4：使用对应技能
├─ 空格：普通攻击
└─ Tab：切换目标

【道具快捷键】
├─ Q：打开道具栏
├─ 数字键：使用对应道具
└─ R：快速使用上一个消耗品

【系统快捷键】
├─ Z：使用时砂
├─ Esc：暂停游戏
├─ I：打开背包
├─ C：打开角色面板
├─ M：打开地图
└─ S：打开商店

【战斗辅助】
├─ T：显示敌人详细信息
├─ G：显示掉落物品
└─ H：隐藏/显示HUD（战斗时）
```

### 8.2 探索快捷键

```
【移动】
├─ WASD/方向键：移动
├─ Shift：冲刺
└─ Ctrl：蹲行（降低被发现概率）

【交互】
├─ E：交互/采集/对话
├─ F：拾取
└─ Tab：切换附近目标

【快速菜单】
├─ J：打开任务面板
├─ K：打开成就面板
└─ L：打开图鉴
```
