### 境界突破流程 (game.gd:303-381)
1. `_show_breakthrough_confirmation()` - 显示确认对话框
2. 用户确认 → `_attempt_breakthrough()`
3. 检查星尘 >= 50，不足则返回据点
4. 扣除50星尘 → 推进境界 → 等级重置为1 → 发送事件

### 数据流总图
```
battle_scene.gd:784 (_on_battle_ended)
    │
    ├─ 奖励计算 (803-818)
    │   ├─ XP = enemy_level * 10 * boss_count

    │   └─ Fragments = 5(精英) / 20*boss_count(BOSS)
    │
    ├─ 势力奖励 (820-840)
    └─ battle_complete.emit(true, rewards)

game.gd:230 (_on_battle_complete)
    │
    ├─ _process_victory(rewards) (239-283)
    │   ├─ RunState.total_xp += xp (246)
    │   ├─ RunState.stardust += stardust (247)
    │   ├─ RunState.add_memory_fragments() (248)
    │   ├─ QuestSystem.notify_battle_complete (251)
    │   ├─ _check_level_up() (274)
    │   ├─ breakthrough_available = is_at_max_level() && !is_max_realm() (277)
    │   └─ zone_complete = cleared >= 4 (262-266)
    │
    └─ _show_victory(...) (394-455)
        ├─ "继续" → _on_victory_continue() → _show_hub()
        ├─ "突破境界" → _show_breakthrough_confirmation()
        └─ "进入下一区域" → _on_zone_advance() → RunState.advance_zone()
```
