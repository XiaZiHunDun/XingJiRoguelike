# Build系统专家评审总结与修改方案

## 五位专家共识

### 核心问题（5位专家全部指出）

| 问题 | 共识度 | 说明 |
|------|--------|------|
| **Build深度不足** | 5/5 | 技能死绑装备，选择空间小；词缀全是数值加成，缺乏联动 |
| **武器技能绑定过死** | 5/5 | 武器3技能全绑，玩家被锁死在武器路线 |
| **元素反应词缀太弱** | 5/5 | 只是伤害倍率，没有机制改变，无法成为Build核心 |
| **装等过多** | 4/5 | 6档太复杂，应精简到3-4档 |
| **套装凑齐太难** | 4/5 | 7槽位凑4件，对Roguelike来说成本过高 |

### 各专家核心建议

**专家1（最详细）：**
- 武器"2+1"模式：1核心+2备选选1携带
- 增加10个"触发型"词缀
- 套装改为2+2结构

**专家2：**
- 词缀分组互斥系统（避免最优解）
- 元素反应词缀机制化
- 诅咒词缀（高风险高回报）

**专家3（最激进）：**
- **技能与装备完全解耦** - 引入技能模组系统
- **灵石重构为"元素赋予器"** - 改变技能属性
- **废除固定ID套装，改为标签共鸣**

**专家4：**
- 元素反应分核心/辅助两类
- 新增"策略类词缀"
- 套装"2+1"结构+元素路线伪套装

**专家5（最犀利）：**
- 风/土系技能缺失，但有对应反应词缀 → 死代码
- 饰品权重倒挂（饰品技能>武器技能）
- 物理系无法触发元素反应，成为孤儿

## 核心修改方案

### 1. 技能来源重构（专家共识：P0）

**当前问题：** 武器3技能全绑，玩家无选择

**推荐方案A（保守）：武器1+2选1**
```
武器 = 1核心技能（固定）+ 2备选技能（战前选择1个携带）
护甲 = 1主动防御 + 1被动特性（被动如"护盾破碎时爆炸"）
饰品 = 1主动 + 1被动词缀槽
总计 = 4个主动技能
```

**推荐方案B（激进-专家3）：技能与装备解耦**
```
引入"技能模组"系统：
- 装备只提供基础属性、词缀、1个被动效果
- 技能作为独立掉落物获取
- 玩家拥有4-6个独立技能槽自由搭配
```

**我的建议：** 采用方案A，Phase 2再考虑方案B

### 2. 词缀系统重构（专家共识：P0）

**当前问题：** 30个词缀全是数值加成，缺乏联动

**重构方案：**

| 类型 | 占比 | 示例 |
|------|------|------|
| 恒定型词缀 | 40% | 火伤+15%、防御+10 |
| 触发型词缀 | 40% | 暴击时附加灼烧、击杀时ATB+20% |
| 代价词缀 | 20% | 伤害+30%但受到伤害+20% |

**新增10个触发型词缀：**
```
【攻击触发类】
- 炽焰之触：暴击时附加灼烧
- 连锁闪电：击杀敌人时闪电链向附近敌人
- 以牙还牙：格挡时反击并ATB+10%

【状态转化类】
- 能量虹吸：能量满时自动释放冲击波
- 动能爆发：动能满时下次技能伤害×1.5

【代价/风险类】
- 玻璃大炮：伤害+30%，受到伤害+50%
- 狂战士：HP<30%时伤害+50%
- 能量过载：能量上限-1，技能伤害+40%
```

**元素反应词缀机制化：**
```
当前：蒸发强化 = 蒸发伤害×1.5
改为：蒸发强化 = 蒸发反应保留火元素，可连续触发

当前：超导强化 = 易伤+15%
改为：超导强化 = 超导反应扩散到周围敌人
```

### 3. 套装系统重构（专家共识：P0）

**当前问题：** 7槽凑4件太难，散件无收益

**方案A：2+2结构（专家1、4建议）**
```
套装改为：2件套（小奖励）+ 4件套（大奖励）
例如磐石：
- 2件：防御+15%
- 4件：受到伤害20%概率免疫+护盾破碎时ATB+30%
```

**方案B：标签共鸣系统（专家3、5最推荐）**
```
废除固定ID套装，改为条件触发：
- 【雷霆万钧】：携带3个雷系词缀 → 触发"雷系攻击20%概率清零ATB"
- 【绝对防御】：携带1护盾技能+1格挡词缀 → 触发"护盾破碎时回复20%HP"

这样任何高词条装备都能融入Build
```

**我的建议：** 采用方案B为主，保留2-3套完整套装作为"大奖"

### 4. 装备随机性优化（专家共识：P1）

**当前问题：** 6档装等太多，词缀浮动过大

**修改方案：**
```
装等：6档 → 4档
- 凡品（绿）：1x属性，1词缀
- 稀有（蓝）：1.5x属性，2词缀
- 史诗（紫）：2x属性，2-3词缀+1特殊词缀
- 传说（橙）：2.5x属性，3词缀+1强力特殊词缀

词缀数值浮动：
- T1-T2：±15%
- T3-T4：±10%

重锻系统改为"附魔圣坛"：
- 献祭一件装备，随机强化另一件的词缀
- 或消耗资源锁定1个词缀后重锻其他词缀
```

### 5. 灵石系统重构（专家共识：P1-P0）

**当前问题：** 3槽位只加10%伤害/速度，无存在价值

**重构方案（专家3最激进）：灵石 = 元素赋予器**
```
当前：火灵石(下) = 火伤+10%, ATB速度+5%
改为：火灵石 = 将技能转化为火属性

示例：
- 镶嵌火灵石后，"横斩(物理)" → "火焰斩(火属性)"
- 镶嵌雷灵石后，"铁壁(护盾)" → "雷电壁垒(受击反弹雷伤)"

这样玩家可以用任何武器走任何元素路线，大幅提升Build灵活性

灵石共振机制：
- 三同元素：元素主宰（灼烧可无限叠加）
- 三不同元素：元素协调（所有反应伤害+30%）
- 两同一异：元素侵蚀（攻击时有机率施加第三种DOT）
```

### 6. 补全缺失内容（专家5指出）

**风/土系技能完全缺失！**

| 元素 | 当前 | 需要 |
|------|------|------|
| 火 | 5技能 ✅ | - |
| 冰 | 4技能 ✅ | - |
| 雷 | 4技能 ✅ | - |
| 风 | 0技能 ❌ | 至少3个技能 |
| 土 | 0技能 ❌ | 至少3个技能 |
| 物理 | 4技能 ✅ | - |

**15种元素反应的6个词缀对应：**
- 风+土 = 沙尘 → 但没有风/土技能，永远无法触发 ❌

**我的建议：** 先砍掉风/土相关词缀，专注火/冰/雷三角；或尽快补充风/土技能

## 最终推荐方案要点

### 技能系统
```
武器 = 1核心技能（固定）+ 2备选技能（战前选择1个）
护甲 = 1主动 + 1被动特性
饰品 = 1主动 + 1被动词缀
灵石 = 被动效果 + 元素赋予（改变技能属性）
总计 = 4主动技能 + 3被动
```

### 词缀系统（30+10个）
```
恒定型（40%）：火伤+15%、防御+10等
触发型（40%）：暴击时附加灼烧、击杀时ATB+20%等
代价型（20%）：伤害+30%但受到伤害+50%等
```

### 套装系统
```
废除固定ID套装 → 改为标签共鸣
- 3个雷系词缀 → "雷霆万钧"效果
- 1护盾技能+1格挡词缀 → "绝对防御"效果
保留2-3套完整套装作为"大奖"
```

### 装备系统
```
4档装等：凡品/稀有/史诗/传说
词缀浮动：±10-15%
重锻：附魔圣坛（献祭装备换取词缀强化）
```
