# 星陨纪元·卡牌ATB Roguelike 设计方案
## 一、设计理念
核心理念：**以「半即时决策压力」为节奏骨架，用「修真元素爆发×科技稳定控制」构建双轨构筑乐趣，让玩家在瞬息万变的战场中，通过卡牌、天赋、装备的组合，实现“科幻械理控场、修真法则破局”的爽感与策略深度并存。**
一句话定位：**不是慢节奏回合制堆数值，也不是纯动作拼反应，而是“停时思考、即时博弈”的科幻修真卡牌构筑。**

---

## 二、系统设计（协同关系）
### 1. ATB半即时战斗核心逻辑
- 敌我共用ATB条，**速度属性决定充能速度**，充能满可行动；
- 玩家**选择卡牌时全局暂停**，释放卡牌后继续计时；
- 怪物无“停时特权”，高攻速怪物会持续施压，逼迫玩家快速决策。

### 2. 卡牌+元素+天赋强协同
- 卡牌自带元素/物理标签，**元素克制直接增伤，元素反应由卡牌组合触发**；
- 修真路线：强化元素叠加、爆发反应、大范围清场（火焚、雷暴、冰域）；
- 科技路线：强化能量循环、护盾、持续控制、炮台/无人机召唤；
- 1+1>2 示例：
  - 科技冰减速控场 → 修真雷元素连锁爆发；
  - 物理破甲 + 土元素护甲粉碎 → 火元素持续灼烧引爆。

### 3. 装备词缀×局内成长
- 装备词缀不单独强数值，而是**强化体系偏向**：
  - 修真向：元素伤害、反应倍率、暴击；
  - 科技向：能量回复、冷却缩减、召唤物强度；
- 词缀与卡牌、天赋形成闭环，避免孤立成长。

### 4. 地图节点与成长节奏
- 战斗/精英提供卡牌/装备；
- 商店定向补强体系；
- 事件提供世界观叙事+特殊机制；
- 休息回复+删卡，保证卡组纯净度。

## 三、数值框架
### 1. ATB与决策窗口
- 基础ATB充能周期：**3~8秒**（随关卡难度提升）；
- 玩家决策窗口：**无强制限时**，但怪物会持续充能，高难度下形成软压力；
- 速度曲线：
  - 小怪：0.8~1.2倍基准速度；
  - 精英：1.1~1.5倍；
  - BOSS：0.7~1.0倍，但技能范围/威胁更大。

### 2. 能量机制
- 基础能量：**3点/回合**，可通过科技天赋/装备堆叠至5~6点；
- 卡牌费用分布：
  - 1费：35%（高频、补伤、过渡）；
  - 2费：45%（核心输出/防御）；
  - 3费：20%（爆发、反应、终结）；
- 无超费溢出惩罚，鼓励**低费循环+高费爆发**交替。

### 3. 元素与克制数值
- 元素克制增伤：**+30%~50%**；
- 元素反应基础倍率：×1.5~×3.0，修真天赋可放大至×4~×6；
- 物理无克制，但享受破甲、暴击、直伤加成。

## 四、内容规模
### 1. 卡牌
- 总卡牌量：**220~260张**；
- 开局初始卡组：12张（基础攻击+防御+少量元素）；
- 中期可收集：60~80张；
- 后期成型卡组：20~28张（通过删卡保持纯净）。

### 2. 天赋
- 双路线：修真/科技，每条**18~22节点**；
- 总天赋节点：40~45个；
- 节点类型：被动强化、主动大招、卡牌强化、机制解锁；
- 支持**混线加点**，拒绝纯单边最优解。

### 3. 装备词缀
- 装备类型：武器、防具、核心、饰品；
- 词缀池：**120~150种**；
- 稀有度：白35%、蓝30%、紫20%、橙10%、红5%；
- 词缀类型：元素强化、能量循环、攻速、护盾、召唤、反应增效。

## 五、风险提示
1. **ATB节奏失控**
   怪物攻速过快导致玩家无决策时间，或过慢退化成回合制；需严格控制速度区间与暂停反馈。
2. **最优解固化**
   某元素+某天赋强度超标，导致全职业同质化；需限制单体系上限，鼓励混搭。
3. **卡牌膨胀与卡组污染**
   卡池过深导致关键卡抓不到，需控制卡池分层与商店定向检索。
4. **科幻修真融合割裂**
   玩法与世界观脱节；需将科技无人机、修真符箓、能量护盾统一视觉与机制。
5. **新手门槛过高**
   ATB+卡牌+元素三系统并行，易造成理解混乱；需分步引导。

## 六、优先级建议
### 第一优先级（核心验证）
1. ATB基础循环 + 暂停出牌机制；
2. 基础能量系统 + 30张核心卡牌；
3. 单路线天赋极简版 + 基础数值。

### 第二优先级（体系成型）
1. 完整元素系统 + 反应机制；
2. 双天赋路线全节点；
3. 装备基础词缀池。

### 第三优先级（体验与内容）
1. 地图节点逻辑与局外成长；
2. 新手引导与死亡反馈；
3. 扩充卡牌、词缀、事件内容。

## 特别关注解答
### 1. ATB 与回合制核心差异
- 回合制：敌我轮流行动，节奏固定、决策无压力；
- 本游戏ATB：**时间永不停止，选择即代价**。
  差异放大方式：
  - 高攻速怪物持续压制；
  - 多目标同时充能，逼迫玩家取舍；
  - 释放高费卡牌时会暴露空档，形成风险收益。

### 2. 科幻×修真在玩法中的体现
- 修真：元素法则、爆发伤害、范围清场、瞬间秒杀；
- 科技：稳定能量、持续控制、护盾、召唤物、循环输出；
- 机制融合：
  - 科技“能量核心”提供稳定蓝量 → 修真“元素大招”瞬间倾泻；
  - 修真“身法提速”→ 科技“高频枪械连射”；
  - BOSS设计：修真法体+科幻机甲，双机制混合挑战。
