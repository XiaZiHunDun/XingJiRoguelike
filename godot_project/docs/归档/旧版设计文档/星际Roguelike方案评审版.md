## 一、游戏概述

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 游戏名称 | 星际Roguelike（星陨纪元） |
| 类型 | Roguelike + ATB半即时战斗 + 开放世界 |
| 平台 | Steam 单机 |
| 核心循环 | 探索 → 战斗 → 掉落装备 → Build构建 → 境界提升 → 死亡重来 |
| 世界观 | 星际旅者坠毁在古代文明遗迹星球，觉醒修炼之路 |

### 1.2 世界设定

```
【背景故事】
玩家是一名坠毁在未知星球的星际旅者。
这个星球曾是繁荣的星际文明的家园，但文明已经衰落。
星球上有5个主要区域，每个区域都有独特的古代文明遗迹。
在遗迹中弥漫着"星尘粒子"，玩家通过吸收星尘粒子觉醒修炼之路。
玩家需要在这些遗迹中探索、收集、建立Build，提升境界，最终找到离开星球的方法。

【阵营势力】
星球上存在多个修炼势力，各有不同的理念和资源。
玩家可以加入某个势力，获得势力独特的装备和任务。
也可以不加入任何势力，自由探索。
势力之间存在竞争关系，击杀对方阵营成员会获得额外奖励。
```

### 1.3 核心体验

```
【传统Roguelike】
拿到什么用什么 → 被动接受随机性

【我们的方案】
看到随机性 → 做决策 → 构建Build → 享受化学反应
```

## 二、战斗系统

### 2.1 ATB半即时战斗

```
【ATB机制】
├─ 速度软上限：200（超出部分转化伤害加成）
├─ 子弹时间：0.1x（代替暂停）
├─ 时砂系统：3次/局，2秒真暂停
└─ 时机加成：完美(≥90%)×1.15，仓促(<70%)×0.8

【战斗流程】
1. ATB蓄力（速度决定蓄力快慢）
2. ATB满 → 进入子弹时间
3. 玩家选择技能
4. 技能释放 → 伤害结算 → ATB重置
5. 循环
```

### 2.2 属性伤害系统

```
【伤害类型】
└─ 物理/奥术

【Phase 1聚焦】物理+奥术双系

【伤害类型特性】
├─ 物理：受防御减免
├─ 奥术：受魔抗减免，能量涌动
└─ 速度溢出：ATB速度超过软上限时，溢出部分转化为伤害加成

【属性联动】（同类型累积强化）
├─ 物理联动：速度溢出高时，物理伤害+15%
└─ 奥术累积：秘法满层 → 技能冷却-20%
```

### 2.3 开放战斗舞台

```
【战斗触发机制】
1. 玩家进入敌人攻击范围
2. 屏幕渐变过渡到战斗区域
3. 玩家和敌人位置保留（战术定位有效）
4. ATB系统启动

【战斗区域特性】
├─ 有边界但保留地图元素（柱子、障碍物）
├─ 地形可能提供战术加成（躲避、元素聚集）
├─ 战斗区域可能有陷阱（持续伤害地面）
└─ 战斗结束后返回地图，敌人消失（死亡/逃跑）

【战术位置影响】
├─ 偷袭（从背后接近）：先手+敌人易伤1回合
├─ 正面对决：正常ATB流程
└─ 被包围（多敌人同时攻击）：敌人ATB加速
```

### 2.4 战术定位系统

```
【先手优势】
├─ 偷袭：先手+敌人易伤1回合
├─ 正面对决：正常流程
└─ 被包围：后手-敌人ATB加速

【战斗区域元素】
├─ 可破坏障碍物
├─ 能量池（站在上面恢复能量）
└─ 危险地形（岩浆地面持续伤害/加速地面提升ATB速度）
```

### 4.6 战斗数值系统

```
【受伤计算流程】

敌方攻击
    ↓
第一步：闪避判定（随机判定是否闪避成功）
    ↓ 闪避成功 → 0伤害
第二步：伤害减免（装备词缀，独立计算）
    ↓
第三步：防御力减伤（减法计算）
    ↓
最终伤害

最终伤害 = (原始伤害 × 伤害减免) - 防御力
```

### 4.7 伤害减免词缀

```
【伤害减免词缀】（仅来自装备词缀）

初级护甲 I-III级
├─ I级：减免5%伤害
├─ II级：减免8%伤害
└─ III级：减免12%伤害

中级护甲 I-III级（按敌人攻击力分段）
├─ I级：敌人攻击力<500时，减免15%
├─ II级：敌人攻击力<1000时，减免20%
└─ III级：敌人攻击力<2000时，减免25%

高级护甲 I-III级（按敌人攻击力分段）
├─ I级：敌人攻击力<1000时，减免30%
├─ II级：敌人攻击力<3000时，减免35%
└─ III级：敌人攻击力<5000时，减免40%

究极护甲 I-III级（敌人攻击力越高减免越多）
├─ I级：每高500攻击力+2%，最高25%
├─ II级：每高500攻击力+3%，最高35%
└─ III级：每高500攻击力+4%，最高45%
```

### 4.8 数值计算公式

```
【HP相关】
HP上限 = 100 + 体质 × 10 + 境界加成 + 装备加成
HP恢复 = HP上限 × (2% + 体质 × 0.1%)

【攻击相关】
物理攻击 = 20 + 体质 × 2 + 装备加成
魔法攻击 = 20 + 精神 × 2 + 装备加成
最终伤害 = 技能倍率 × 对应攻击 × (1 + 增伤词缀)

【防御相关】
物理防御 = 体质 × 1.5 + 装备加成
魔法防御 = 精神 × 1.5 + 装备加成
伤害减免 = 防御 / (防御 + 200) × 100%

【能量相关】
能量上限 = 5 + floor(体质 / 20)
能量恢复 = 2 + floor(体质 / 25) 点/回合

【速度相关】
ATB速度 = 100 + 敏捷 × 3 + 装备加成
速度溢出：ATB速度超过200时，超出部分获得额外效果（伤害+1%每点溢出）

【闪避相关】
闪避率 = min(50%, 敏捷 × 0.4% + 装备加成)

【暴击相关】
暴击率 = min(80%, 敏捷 × 0.5% + 装备加成)
暴击伤害 = 150% + 敏捷 × 0.2% + 装备加成
```

## 五、装备系统

### 5.1 核心设计

```
【取消】
├─ 稀有度系统（凡品/稀有/史诗/传说）
└─ 装等系统（T1-T6）

【替代】穿戴要求系统
├─ 装备掉落时随机生成穿戴要求
├─ 玩家属性/技能/等级满足要求才能穿戴
└─ 高要求 = 通常意味着高收益
```

### 5.2 掉落随机性

```
【技能】（每件装备，各20%均匀分布）
├─ 0技能：20%（纯词缀装）
├─ 1技能：20%
├─ 2技能：20%
├─ 3技能：20%
└─ 4技能：20%

【词缀】
├─ 数量：1-4个
└─ 类型：从词缀池随机

【穿戴要求】（三维随机组合）
├─ 属性要求（体质/精神/敏捷）
├─ 境界要求（对应装备境界等级）
└─ 技能要求（特定技能数量）
```

### 5.3 穿戴要求系统

```
【属性要求】（随机1-2种组合）
├─ 体质要求 >= X（5/10/15/20...）
├─ 精神要求 >= X（5/10/15/20...）
├─ 敏捷要求 >= X（5/10/15/20...）
└─ 境界要求（凡人境/感应境/聚尘境等）

【技能要求】（随机1种）
├─ 终结技等级 >= X（1/2/3）
└─ 物理/魔法技能数量 >= X

【穿戴规则】
├─ 必须同时满足属性要求和境界要求
└─ 技能要求为可选，满足可获得额外加成
```

### 5.4 装备掉落示例

```
【感应境战斗掉落】

奥法法杖
├─ 境界要求：感应境及以上
├─ 属性要求：精神>=20
├─ 技能：奥术弹(2)、能量汲取(2)
```

## 七、词缀联动系统

### 7.1 联动类型

```
【装备↔角色联动】
- 形态改变词缀 → 改变角色技能形态
- 魔法增强词缀 → 强化角色魔法技能
- 触发型词缀 → 角色技能触发装备词缀效果

【装备↔装备联动】
- 物理攻击装备 + 物理攻击装备 → 物理伤害加成
- 魔法攻击装备 + 魔法攻击装备 → 魔法伤害加成
- 多件装备词缀同时满足条件 → 触发额外效果

【词缀↔词缀联动】
- 同时拥有"奥术精通"和"法术暴击" → 奥术效果最大化
- 同时拥有"能量涌动"和"暴击回能" → 能量循环
```

### 7.3 玩家技能装配

```
【技能装配系统】
├─ 角色固定技能：4个（始终可用）
├─ 装备技能：0-4个/装备（自动解锁）
├─ 常用技能栏：玩家手动装配3-4个常用技能
└─ 战斗快捷键：1-4对应常用技能，5-8对应其他技能

【操作设计】
├─ 玩家通过技能装配界面选择3-4个常用技能
├─ 战斗中按快捷键释放，不受技能总数影响
└─ 解决了"技能过多导致选择瘫痪"的问题
```

## 十八、敌人系统

### 18.1 敌人分类

```
【普通敌人：原生异兽】

【BOSS：区域之主】
├─ 每区域1个，代表区域终极威胁
├─ 独特的阶段和机制
├─ 击败后解锁该区域核心内容和主线推进
```

```
【支线任务链】
├─ 每区域2-3条支线任务链
├─ 每条任务链有3-5个任务
```

```
├─ 击杀触发：击败特定敌人
```

### 12.1 资源类型

```
【星尘】
├─ 来源：击杀敌人、任务奖励、卖出装备

├─ 来源：BOSS掉落、特殊事件
├─ 用途：局外永久解锁、兑换唯一性装备图纸
└─ 定位：多周目动力

├─ 来源：采集、击杀敌人、任务奖励
├─ 用途：制作消耗品、装备升级、任务道具
└─ 定位：局内探索与战斗辅助

【增幅精华】
├─ 来源：BOSS掉落、任务奖励、记忆碎片兑换
```

## 二十六、总结

**核心循环：**
```
选择角色（定Identity）
    ↓
探索区域（自由选择路线）
    ↓
遭遇敌人（战术定位）
    ↓
开放战斗（ATB+属性联动）
    ↓
战斗掉落（惊喜盒）
    ↓
检查穿戴要求（能否穿上？）
    ↓
决策（换/不换/保留）
    ↓
解锁技能变体（局内成长）
    ↓
词缀联动（化学反应）
    ↓
完成任务链（主线/势力/支线）
    ↓
提升境界（属性/境界突破）
    ↓
加入阵营（贡献/兑换独特物资）
    ↓
解锁更多区域/内容
    ↓
循环直到通关或死亡
```

**V5方案的关键设计：**

### 角色与境界
1. **多角色选择** - 星际战士/奥术师，各有独特技能
2. **境界等级** - 11境界（凡人境→星火境），每境10级，境界即等级
3. **属性系统** - HP/攻击/防御/精神/敏捷/幸运，丰富Build维度
4. **技能变体** - 局内解锁变体，技能形态多样

### Build系统
5. **装备是主体** - 技能均匀分布(各20%)，玩家主要精力所在
6. **穿戴要求** - 境界要求+三维随机，次要加成通过词缀实现
7. **词缀联动** - 反应型30%作为核心，形态改变15%增加深度
8. **代价分级** - 新手级/专家级代价，降低新手门槛
9. **联动保底** - 1-2个同系词缀也能触发基础共鸣，保证Build下限
10. **属性联动** - 物理/奥术属性联动成为Build核心循环
11. **唯一性装备** - 世界唯一，6件独特装备，追求终极目标

### 阵营势力
12. **四大阵营** - 星火殿/寒霜阁/机魂教/守墓人，各有独特物资
13. **贡献值系统** - 完成任务/击杀守墓人获取，兑换独特装备
14. **势力敌人** - 野外随机刷新守墓人成员，击杀掉落势力装备
15. **阵营关系** - 唯一阵营加入，守墓人唯一敌对

### 开放世界
12. **5个文化差异区域** - 沙海回声/霜棘王庭/翠蔓圣所/机魂废土/太初核心
13. **传送门连接** - 区域间瞬移，探索无阻碍
14. **主线不强制** - 玩家可自由探索，势力任务引导地图进度

### 敌人与战斗
15. **原生异兽+古代守卫** - 普通/精英/BOSS三级敌人
16. **势力野怪** - 各势力成员野外刷新，掉落势力装备
17. **开放战斗舞台** - 战术定位保留，地形战术互动
18. **遭遇站位战术** - 偷袭先手/正面对决/被包围
