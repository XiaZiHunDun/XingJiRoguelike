## 1. 游戏概述

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 游戏名称 | 星际Roguelike（星陨纪元） |
| 类型 | Roguelike + ATB半即时战斗 + 装备驱动Build |
| 平台 | Steam 单机 |
| 核心循环 | 探索 → 战斗 → 掉落装备 → Build构建 → 境界提升 → 死亡重来 |

### 1.2 世界设定

**背景故事**
```
玩家是一名坠毁在未知星球的星际旅者。
这个星球曾是繁荣的星际文明的家园，但文明已经衰落。
星球上有5个主要区域，每个区域都有独特的古代文明遗迹。
在遗迹中弥漫着"星尘粒子"，玩家通过吸收星尘觉醒修炼之路。
玩家需要在遗迹中探索、收集，建立Build，提升境界，最终找到离开星球的方法。
```

### 1.3 核心体验

```
【传统Roguelike】
拿到什么用什么 → 被动接受随机性

【我们的方案】
看到随机性 → 做决策 → 构建Build → 享受化学反应
```

## 2. 战斗系统

### 2.1 ATB机制

```
【ATB机制】
├─ ATB槽：每个角色和敌人有一个蓄力槽
├─ 蓄力速度 = 100 + 敏捷 × 3 + 装备加成
├─ 槽满即可行动（普攻/技能/道具/逃跑）
├─ 速度软上限200，超出转化伤害加成
├─ 子弹时间：槽蓄满前一刻触发0.1x慢动作
└─ 时砂系统：3次/局，2秒真暂停

【行动类型】
├─ 攻击：使用普攻或技能
├─ 技能：消耗能量释放强力技能
├─ 道具：使用消耗品
└─ 逃跑：放弃战斗（可能损失掉落）
```

### 2.2 伤害类型

```
【伤害类型】
├─ 物理：受防御减免
├─ 奥术：受魔抗减免

【伤害公式】
最终伤害 = (原始伤害 × 伤害减免) - 防御/魔抗
```

### 2.3 关键数值

```
【闪避率】min(50%, 敏捷×0.4% + 装备加成)
【暴击率】min(80%, 敏捷×0.5% + 装备加成)
【暴击伤害】150% × (1 + 敏捷×0.2% + 装备加成%)
【速度溢出】ATB速度>200时，超出部分×1%伤害加成
```

## 5. 装备系统

### 5.1 核心设计

```
【取消】
├─ 稀有度系统（凡品/稀有/史诗/传说）
├─ 装等系统（T1-T6）
└─ 装备分解系统

【替代】
├─ 穿戴要求系统：代替装等/稀有度作为装备门槛
├─ 只能出售：装备作为资源而非燃料
└─ 技能+词缀随机：每件装备独特
```

### 5.2 技能分布

```
【技能数量分布】（各20%均匀分布）
├─ 0技能：20%（纯词缀装）
├─ 1技能：20%
├─ 2技能：20%
├─ 3技能：20%
└─ 4技能：20%
```

### 5.3 穿戴要求

```
【要求类型】
├─ 属性要求：体质/精神/敏捷达标
├─ 境界要求：对应境界等级
└─ 技能要求：特定技能等级

【穿戴规则】
├─ 全部满足才能穿戴
├─ 不满足则不可穿戴，但可出售
└─ 不可穿戴装备不影响其他装备
```

### 5.4 装备类型

```
【武器】
├─ 巨剑：物理攻击，体质×1.5+10
├─ 法杖：魔法攻击，精神×1.5+10
└─ 双刃：物理暴击，敏捷×1.2+8

【护甲】
├─ 钢甲：高防御，高体质
├─ 法袍：高魔抗，高精神
├─ 皮甲：高速度，高敏捷
└─ 修真袍：综合

【饰品】（2个槽位）
├─ 提供各种词缀效果
└─ 戒指/项链/护符等
```

### 5.5 增幅器系统

```
【增幅器】（3个格子，按境界解锁）
├─ 体质增幅器：体质×1.5/2.0/2.5
├─ 精神增幅器：精神×1.5/2.0/2.5
├─ 敏捷增幅器：敏捷×1.5/2.0/2.5
├─ HP增幅器：HP×1.3/1.6/2.0
├─ 魔法攻击增幅器：魔法攻击×1.3/1.6/2.0
├─ 防御增幅器：防御×1.3/1.6/2.0
└─ 速度增幅器：ATB速度×1.3/1.6/2.0

【品质等级】
├─ 绿色（初）：基础倍率
├─ 蓝色（中）：中级倍率
└─ 紫色（极）：高级倍率
```

### 10.2 加入规则

```
【加入条件】
├─ 玩家可选择加入任意一个阵营
├─ 加入后无法退出（永久绑定）
└─ 同时只能拥有一个阵营身份
```

### 10.3 阵营关系

```
【关系等级】（初始值50/100）
├─ 友好（80-100）：商店价格正常
├─ 中立（50-79）：商店价格+20%
├─ 冷淡（20-49）：商店价格+50%
└─ 敌对（0-19）：商店价格+100%

【关系变化】
├─ 攻击该阵营成员：-5~-20
├─ 击杀该阵营成员：-10~-30
└─ 完成该阵营任务：+5~+15
```

### 10.4 贡献等级

| 等级 | 贡献值范围 | 解锁内容 |
|------|------------|----------|
| 学徒 | 0-100 | 基础装备 |
| 正式成员 | 101-500 | 中级装备 |
| 精英成员 | 501-1500 | 高级装备、稀有词缀 |
| 长老 | 1501-5000 | 终极装备、绿色增幅器 |
| 掌门 | 5000+ | 蓝色/紫色增幅器、唯一性装备图纸 |

### 10.5 每日任务

```
【任务池规则】
├─ 每个阵营有独立的20个任务池
├─ 每日凌晨刷新20个任务
└─ 玩家只能看到自己阵营的任务

【任务类型】
├─ 击杀任务、收集任务
├─ 探索任务、挑战任务
└─ 难度：简单/普通/困难/噩梦
```

## 13. 数值框架

### 13.1 角色成长

```
【每级属性成长】
├─ 体质：+3/级
├─ 精神：+2.5/级
├─ 敏捷：+2.5/级
├─ HP：+8/级
└─ 能量：+0.5/级

【50级基础属性】
├─ 体质：187，物理攻击：964
├─ 精神：142，魔法攻击：745
├─ 敏捷：142，ATB速度：526
├─ HP：1696
└─ 防御：355
```

### 13.2 敌人数值

```
【50级敌人】
├─ 普通怪：HP 1050，攻击260
├─ 精英怪：HP 3150，攻击390
└─ BOSS：HP 10500，攻击520

【HP倍率】
├─ 普通怪：1x
├─ 精英怪：3x
└─ BOSS：10x
```

### 13.3 战斗伤害计算

```
【物理伤害】
最终伤害 = (物理攻击 × 技能倍率 × 时机加成) × (1-减伤比例) - 防御

【魔法伤害】
最终伤害 = (魔法攻击 × 技能倍率 × 时机加成) × (1-减伤比例) - 魔抗

【防御减伤】
减伤比例 = 防御 / (防御 + 100)
```

### 13.4 战斗时长目标

| 战斗类型 | 击杀下数 | 战斗时长 |
|----------|----------|----------|
| 普通怪 | 3-5下 | 6-15秒 |
| 精英怪 | 8下 | 16秒 |
| BOSS | 25下 | 50秒 |
