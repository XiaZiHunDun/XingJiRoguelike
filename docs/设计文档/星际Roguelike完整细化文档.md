## 第一章 游戏概述

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 游戏名称 | 星际Roguelike（星陨纪元） |
| 类型 | Roguelike + ATB半即时战斗 + 装备驱动Build |
| 平台 | Steam 单机 |
| 核心循环 | 探索 → 战斗 → 掉落装备 → Build构建 → 境界提升 → 死亡重来 |

### 1.2 世界设定

```
背景故事：
玩家是一名坠毁在未知星球的星际旅者。这个星球曾是繁荣的星际文明的家园，但文明已经衰落。
星球上有5个主要区域，每个区域都有独特的古代文明遗迹。在遗迹中弥漫着"星尘粒子"，
玩家通过吸收星尘觉醒修炼之路。玩家需要在遗迹中探索、收集，建立Build，提升境界，最终找到离开星球的方法。

阵营势力：
星球上存在多个修炼势力，各有不同的理念和资源。玩家可以加入某个势力，获得势力独特的装备和任务。
玩家同时只能加入一个阵营。守墓人是唯一敌对阵营，击杀可获得贡献。
```

### 1.3 核心体验

```
【传统Roguelike】
拿到什么用什么 → 被动接受随机性

【我们的方案】
看到随机性 → 做决策 → 构建Build → 享受化学反应
```

## 第二章 战斗系统

### 2.1 ATB半即时战斗

```
【ATB机制】
├─ ATB槽：每个角色和敌人有一个蓄力槽
├─ 蓄力速度 = 100 + 敏捷 × 3 + 装备加成
├─ 槽满即可行动（普攻/技能/道具/逃跑）
├─ 速度软上限300，超出转化伤害加成(0.5% per point over cap)
├─ 子弹时间：完美时机(≥90%)触发0.2x慢动作
└─ 时砂系统：初始2次，每击杀5个敌人恢复1次，上限3次，2秒真暂停

【行动类型】
├─ 攻击：使用普攻或技能
├─ 技能：消耗能量释放强力技能
├─ 道具：使用消耗品
└─ 逃跑：放弃战斗（可能损失掉落）

【完美时机】
├─ 完美(≥90%) × 1.15
├─ 良好(70-89%) × 1.0
└─ 仓促(<70%) × 0.8
```

### 2.2 属性伤害系统

```
【伤害类型】
├─ 物理：受防御减免
└─ 奥术：受魔抗减免

【伤害公式】
最终伤害 = (原始伤害 × 伤害减免) - 防御/魔抗

【受伤计算流程】
敌方攻击 → 闪避判定 → 伤害减免 → 防御减伤 → 最终伤害
```

### 2.3 战斗数值

```
【闪避率】min(50%, 敏捷 × 0.4% + 装备加成)
【暴击率】min(80%, 敏捷 × 0.5% + 装备加成)
【暴击伤害】150% × (1 + 敏捷 × 0.2% + 装备加成%)
【ATB速度】100 + 敏捷 × 3 + 装备加成
【速度溢出】ATB速度>300时，超出部分×0.5%伤害加成
```

### 2.4 能量系统

```
【能量恢复】
├─ 初始能量：5点
├─ 最大能量：5 + 精神 × 0.5 + 装备加成
├─ 每回合结束：+0.5能量
├─ 暴击时（需"暴击回能"词缀）：+1能量
└─ 击杀敌人：+0.5能量
```

## 第五章 装备系统

### 5.1 核心设计

```
【取消】
├─ 稀有度系统（凡品/稀有/史诗/传说）
├─ 装等系统（T1-T6）
└─ 装备分解系统

【替代】
├─ 穿戴要求系统：代替装等/稀有度作为装备门槛
├─ 只能出售：装备作为资源而非燃料
└─ 技能+词缀随机：每件装备独特
```

### 5.2 技能分布

```
【技能数量分布】（各20%均匀分布）
├─ 0技能：20%（纯词缀装）
├─ 1技能：20%
├─ 2技能：20%
├─ 3技能：20%
└─ 4技能：20%
```

### 5.3 穿戴要求

```
【要求类型】
├─ 属性要求：体质/精神/敏捷达标
├─ 境界要求：对应境界等级
└─ 技能要求：特定技能等级

【穿戴规则】
├─ 全部满足才能穿戴
├─ 不满足则不可穿戴，但可出售
└─ 不影响其他装备

【示例】
装备A：体质≥25要求 → 玩家体质25可穿戴
装备B：体质≥30要求 → 玩家体质25不可穿戴，可出售
```

### 5.4 装备类型

```
【武器】
├─ 巨剑：物理攻击，体质×1.5+10
├─ 法杖：魔法攻击，精神×1.5+10
└─ 双刃：物理暴击，敏捷×1.2+8

【护甲】
├─ 钢甲：高防御，高体质
├─ 法袍：高魔抗，高精神
├─ 皮甲：高速度，高敏捷
└─ 修真袍：综合

【饰品】（2个槽位）
└─ 提供各种词缀效果

【装备等级来源】
└─ 装备等级 = 所在小地图的怪物等级
```

### 5.5 增幅器系统

```
【增幅器】（3个格子，按境界解锁）
├─ 体质增幅器：体质×1.5/2.0/2.5
├─ 精神增幅器：精神×1.5/2.0/2.5
├─ 敏捷增幅器：敏捷×1.5/2.0/2.5，ATB+10%/20%/30%
├─ HP增幅器：HP×1.3/1.6/2.0
├─ 魔法攻击增幅器：魔法攻击×1.3/1.6/2.0
├─ 防御增幅器：防御×1.3/1.6/2.0
└─ 速度增幅器：ATB速度×1.3/1.6/2.0

【品质等级】
├─ 绿色（初）：基础倍率
├─ 蓝色（中）：中级倍率
└─ 紫色（极）：高级倍率

【获取方式】
├─ 贡献兑换：绿色300/蓝色600/紫色1200
├─ 任务奖励
├─ BOSS掉落
└─ 特殊事件

【特性】
├─ 科技装备，不带技能和词缀
├─ 不可出售
├─ 不可重铸
├─ 可随时更换
└─ 死亡后保留
```

### 5.6 唯一性装备

```
【世界唯一装备】
├─ 同一时间整个世界只有一件
├─ 死亡后消失，可再次获取
├─ 获取方式：任务奖励/成就/守墓人兑换/极低概率掉落
└─ 共6件

【列表】
1. 星陨剑：暴击触发陨石
2. 虚空护符：死亡保留50%星尘
3. 寒霜之心：攻击减速敌人

5. 星火战甲：受伤火焰伤害
```

```
【50级敌人】
```

### 13.4 战斗伤害计算

```
【物理伤害】
最终伤害 = (物理攻击 × 技能倍率 × 时机加成) × (1-减伤比例) - 防御

【魔法伤害】
最终伤害 = (魔法攻击 × 技能倍率 × 时机加成) × (1-减伤比例) - 魔抗

【防御减伤】
减伤比例 = 防御 / (防御 + 100)
```

### 13.5 战斗时长目标

| 精英怪 | 850 | 3150 | 8下 | 16秒 | 10-15秒 |
| BOSS | 850 | 10500 | 25下 | 50秒 | 30-40秒 |

| 战斗时长目标 | BOSS 50秒 | **BOSS 30-40秒** | Roguelike节奏优化 |
