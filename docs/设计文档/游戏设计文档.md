> **文档版本**：4.1
> **创建日期**：2026-04-13
> **文档状态**：**装备驱动型重构**（基于5位专家评审建议修订）

## 一、游戏愿景与核心理念

### 1.1 核心理念

**「以器证道，器法通天」**

> **"在时间与命运的双重博弈中，装备是你的第二生命。"**

**装备即Build**：不像传统Roguelike以卡牌或天赋为核心，本作的Build核心是**装备**。卡牌是装备召唤的技能，天赋是装备的强化配件。

**时间即资源**：ATB蓄力条持续前进，玩家在"等待完美时机"和"抢先出手"之间做抉择——这是回合制无法提供的紧张感。

**决策即博弈**：不是"我回合做什么"，而是"现在出牌值不值，还是再等等"——每次决策都有机会成本。

### 1.2 游戏类型重新定义

| 维度 | 原定位（卡牌驱动） | **新定位（装备驱动）** |
|------|-------------------|----------------------|
| **核心Build** | 卡牌组合 | **装备词缀** |
| **技能表现** | 卡牌本身 | **卡牌是装备的技能表现** |
| **成长来源** | 卡牌获取/升级 | **装备词缀/套装** |
| **天赋作用** | 独立Build分支 | **装备的强化/辅助** |
| **参考游戏** | 《杀戮尖塔》 | **《侥幸骑士》+《暗黑破坏神》** |

### 1.3 科幻×修真的玩法体现

| 世界观概念 | 玩法映射 |
|-----------|---------|
| 灵气/科技冲突 | **装备双路线**：修真装备 vs 科技装备 |
| 突破境界 | **装备稀有度突破**（白→绿→蓝→紫→橙→红） |
| 精密仪器 | **词缀系统**（随机生成 + 套装效果） |
| 法宝/武器 | **核心装备槽**（武器+护甲+饰品） |

### 1.4 ATB vs 回合制的核心差异

| 维度 | 传统回合制 | **星陨纪元ATB** |
|------|-----------|----------------|
| 决策时间 | 无限 | **有限（6-8秒固定窗口）** |
| 时间压力 | 无 | **持续紧迫感** |
| 暂停机制 | 无 | **子弹时间（0.1x），非完全冻结** |
| 核心乐趣 | 策略深度 | **时机博弈+策略深度** |

## 二、ATB战斗系统设计（修订版）

### 2.1 核心机制：子弹时间 + 行动队列

**关键改变**：取消"选牌时全局暂停"，改为子弹时间机制。

```
【子弹时间系统】

选牌界面开启时：
├── 全局ATB降至 0.1x（极慢而非冻结）
├── 玩家始终能感受到敌人ATB在逼近
└── 屏幕边缘敌人ATB条高亮闪烁（威胁预警）

时砂资源（可选消耗）：
├── 每局初始3点时砂
├── 消耗1点可进入真正暂停（3秒）
├── 时砂耗尽后只能0.1x慢速选牌
└── 击杀敌人可掉落时砂（精英/BOSS必掉）
```

**行动队列机制**：
```
选牌界面中：
├── 玩家可预先排列最多3张卡牌
├── 卡牌进入"行动队列"而非立即结算
├── ATB满时按队列顺序执行所有动作
└── 执行过程中可继续累积下一轮ATB
```

### 2.2 ATB速度曲线（修订版）

**决策窗口固定化**：决策窗口不再随速度变化，固定为6-8秒。

| 档位 | 速度值 | 满槽时间 | 行动频率 | 风格定位 |
|------|--------|----------|----------|----------|
| 1 | 3点/秒 | 33秒 | 极低 | 龟速Build |
| 3 | 6点/秒 | 16秒 | 低 | 策略深度 |
| 5 | 10点/秒 | 10秒 | 中 | 平衡点 |
| 7 | 15点/秒 | 6.6秒 | 高 | 快节奏 |
| 10 | 25点/秒 | 4秒 | 极高 | 极限反应流 |

**公式**：`满槽时间(秒) = 10 / (1 + 速度/100)`
- 速度100时=5秒，速度200时=3.3秒
- 边际递减防止极端Build

**速度来源**：
- 基础速度由角色属性决定
- **装备词缀可额外增加速度**（如："ATB速度+15%"）
- 灵石槽的灵石也可提供速度（如：风灵石+10%速度）

**速度软上限**：超过速度200后，收益转化为"动能"（下次攻击伤害+10%），而非继续加快ATB。

### 2.3 临界点机制（保留）

```
【蓄力阶段】（0-70% ATB）
- 可随时出牌，但威力-20%（仓促惩罚）
- 敌人ATB条高亮闪烁

【临界点】（70-100% ATB）
- 正常威力出牌
- 90%时出牌触发「完美时机」，伤害+15%

【溢出风险】（100% ATB后）
- 改为"ATB冻结"而非扣血
- 冻结期间无法行动，但仍可选择下一轮卡牌
```

### 2.4 修真/科技路线ATB差异

| 路线 | ATB特权 | 机制 |
|------|----------|------|
| **修真** | 可透支时砂（进入负值） | 透支1点，本局最大HP-10%（体现"突破代价"） |
| **科技** | 时砂自然恢复 | 每2分钟恢复1点（体现"可持续稳定"） |

## 三、元素系统设计（修订版）

### 3.1 六元素定位

**元素来源于装备**：武器决定技能的元素属性，灵石槽提供元素亲和。

| 元素 | 核心特性 | 异常状态 | **ATB关联（关键）** |
|------|---------|---------|---------------------|
| **火** | 爆发、DOT | 灼烧：ATB恢复-30% | ✅ 直接影响ATB |
| **冰** | 控制、减速 | 冻结：ATB暂停2秒 | ✅ 直接影响ATB |
| **雷** | 连锁、破防 | 感电：被攻击时ATB倒退10% | ✅ 新增关联 |
| **风** | 位移、增益 | 风缠：行动后额外+10%ATB | ✅ 新增关联 |
| **土** | 耐久、护盾 | 石化：ATB增长速度-30% | ✅ 新增关联 |
| **物理** | 稳定、破盾 | 破甲：防御-50% | 无ATB关联 |

### 3.2 元素与装备的关联

```
元素来源：
├── 武器元素：剑（火剑）/ 法杖（雷杖）/ 枪械（冰弹）
├── 灵石槽：3个灵石槽决定元素亲和
└── 饰品词缀：可添加额外元素效果

元素Build搭配：
├── 火剑 + 火灵石×3 = 纯火Build（伤害爆发）
├── 冰杖 + 风灵石×2 = 冰风控制（减速辅助）
└── 物理枪 + 土灵石×2 = 物理防御（护盾流）
```

### 3.3 元素克制关系（简化）

```
简化原则：克制效果+20%，被克制-10%（原+30%/-15%过于强力）
```

### 3.4 元素反应（补全15种二阶反应）

**所有反应都与ATB相关**：

| 组合 | 反应名称 | ATB效果 |
|------|----------|---------|
| 火+风 | 焚风 | 灼烧扩散，敌人ATB恢复-50% |
| 火+冰 | 蒸发 | 伤害×2，清除双方异常状态 |
| 火+雷 | 烈焰雷霆 | 灼烧目标被雷击时爆炸，ATB倒退20% |
| 火+土 | 熔岩 | 地面持续伤害2秒，命中-20% |
| 冰+雷 | 超导 | 易伤+25%，减速50%，ATB-15% |
| 冰+风 | 寒流 | 全场减速，ATB速度-20% |
| 冰+土 | 冻土 | 定身1秒，ATB冻结0.5秒 |
| 雷+风 | 电离 | 额外闪电连锁，速度+20%持续 |
| 雷+土 | 磁化 | ATB吸引，周围敌人ATB向你靠拢 |
| 风+土 | 沙暴 | 致盲（50%未命中），持续掉血 |
| 火+火(自) | 炽焰 | 连续灼烧，伤害递增20% |
| 冰+冰(自) | 绝对零度 | 冻结时间延长至4秒 |
| 雷+雷(自) | 雷电蓄积 | 连续雷击，伤害递增15% |
| 风+风(自) | 风起云涌 | 闪避率+30%，ATB+10% |
| 土+土(自) | 大地震颤 | 护盾+50%，反击伤害+30% |

**元素连携（替代三阶反应）**：
- 删除复杂三阶反应
- 连续释放3个不同元素技能后，触发"元素共鸣"（全体敌人易伤20%，持续3秒）
- 更易触发，鼓励元素混合Build

**元素反应与装备联动示例**：
- 【火剑】技能"火焰斩" + 【雷灵石】→ 触发"烈焰雷霆"
- 【冰杖】技能"寒冰箭" + 【风灵石】→ 触发"寒流"

**元素反应分阶段解锁**（专家建议）：
```
第一章：仅出现3种基础反应（火+风、冰+雷、雷+风）
第二章：增加5种反应
第三章：全部15种反应
UI提示：战斗界面显示"当前元素组合预览"，鼠标悬停敌人显示"可触发反应"

## 四、装备系统设计（核心）

### 4.1 装备定位重构

**装备是Build的核心**，不是词缀的附属品。

```
【装备系统核心理念】

传统设计：装备 = 数值提供器（攻击+10，防御+5）
本作设计：装备 = 技能+规则改变器

每件装备都携带：
├── 基础属性（攻防血）
├── 1-3个词缀（规则改变）
└── 套装效果（多件激活）
```

### 4.2 装备槽位设计

| 槽位 | 数量 | 核心定位 | 可装备类型 |
|------|------|----------|-----------|
| **武器** | 1 | 主要输出+元素属性 | 剑/法杖/枪械/灵宠 |
| **护甲** | 1 | 防御+抗性 | 铠甲/长袍/护盾 |
| **饰品** | 2 | 特殊效果+Build核心 | 戒指/项链/护符 |
| **灵石** | 3 | 元素亲和+ATB调节 | 火/冰/雷/风/土/物理 |

**灵石槽优化**：
- 灵石有**独立背包**（不占用装备背包格）
- 灵石可升级（低品质灵石可合成为高品）

### 4.3 装备稀有度体系

| 稀有度 | 名称 | 词缀数 | 品质定位 |
|--------|------|--------|----------|
| 白色 | 凡品 | 1 | 过渡装 |
| 绿色 | 精品 | 1-2 | 基础Build |
| 蓝色 | 极品 | 2 | 完整Build |
| 紫色 | 史诗 | 2-3 | 强化Build |
| 橙色 | 传说 | 3 | 毕业追求 |
| 红色 | 神话 | 3+限定词缀 | 极限Build |

### 4.4 词缀系统（核心Build机制）

**每件装备1-3个词缀，词缀决定Build方向**：

| 词缀类型 | 效果示例 | Build定位 |
|---------|---------|-----------|
| **伤害词缀** | 火属性伤害+15% | 元素爆发Build |
| **ATB词缀** | 攻击时10%概率清零目标ATB | 控制型Build |
| **能量词缀** | 能量上限+2 | 高费技能Build |
| **元素反应词缀** | 蒸发伤害×2.5 | 元素反应Build |
| **特殊词缀** | 击杀敌人恢复HP | 续航Build |
| **套装词缀** | 2件套：ATB速度+20% | 套装流 |

**词缀分层体系**（专家建议）：

| 层级 | 数量 | 定位 | 示例 |
|------|------|------|------|
| **核心词缀** | 约20个 | 决定Build基础方向 | ATB速度+15%、火伤+20% |
| **扩展词缀** | 约30个 | 丰富Build选择 | 击杀回血、能量恢复+10% |
| **稀有词缀** | 约10个 | 提供极端Build选项 | 完美时机伤害+50%、ATB清零 |

**词缀平衡规则**：
- **同类词缀软上限**：同源词缀（如多个ATB速度词缀）最多叠加3层，超出部分转化为"动能"
- **词缀互斥**：某些强力词缀不能共存（如"ATB冻结"与"ATB加速"）
- **词缀50%以上为规则改变型**，而非纯数值型

### 4.5 套装效果系统

**集齐套装激活额外效果，鼓励追求特定组合**：

```
【示例套装】

"量子丹田"（科技路线）
├── 2件：能量恢复+30%
├── 3件：释放技能时ATB+15%
└── 4件：能量溢出转为下次技能伤害+50%

"雷劫"（修真路线）
├── 2件：雷系伤害+25%
├── 3件：雷系攻击15%概率重置ATB
└── 4件：完美时机触发雷劫，伤害×3
```

**套装保底机制**：
- **套装碎片系统**：掉落装备时概率获得该套装碎片，集齐合成
- **凑套减压**：2件套=60%完整效果，3件套=80%，4件套=100%
- **保底掉落**：每击败3个精英，必掉1件"当前装备栏位"的装备

### 4.6 装备获取与强化

| 阶段 | 获取方式 | 强化方式 |
|------|---------|---------|
| **战斗中** | 击败敌人掉落 | 无 |
| **商店** | 消耗金币购买 | 用金币强化词缀 |
| **锻造** | 消耗材料制作 | 词缀重随（可选锁定1条） |
| **事件** | 随机事件奖励 | 套装组件 |

**商店定向刷新**（专家建议）：
- 每章商店刷新1件"指定词缀"装备（价格+50%）
- 帮助玩家定向Build，解决随机性过强问题

| 阶段 | 获取方式 | 强化方式 |
|------|---------|---------|
| **战斗中** | 击败敌人掉落 | 无 |
| **商店** | 消耗金币购买 | 用金币强化词缀 |
| **锻造** | 消耗材料制作 | 词缀重随（保底机制） |
| **事件** | 随机事件奖励 | 套装组件 |

## 五、技能系统（卡牌的表现方式）

### 5.1 技能 vs 卡牌重新定义

**卡牌是装备技能的"释放方式"，而非独立Build系统**。

```
技能来源：装备（武器/护甲/饰品）决定你有哪些技能
卡牌表现：手牌是技能的快捷释放方式

举例：
- 【火剑】装备 → 提供技能"火焰斩"
- 手牌中有"火焰斩"卡牌 → 这是该技能的释放形式
- 装备词缀"火焰斩伤害+20%" → 影响的是技能效果
```

### 5.2 技能类型（按来源）

| 技能来源 | 数量 | 特点 |
|---------|------|------|
| **武器技能** | 3-5个/武器 | 主要输出，消耗能量 |
| **护甲技能** | 2-3个/护甲 | 防御/反制 |
| **饰品技能** | 1-2个/饰品 | 特殊效果 |
| **通用技能** | 10-15个 | 无装备限制 |

### 5.3 卡牌设计简化

卡牌设计聚焦于**技能释放的策略性**，而非Build核心：

| 类型 | 定位 | 费用特征 |
|-----|------|---------|
| **攻击技能** | 伤害输出 | 1-3费 |
| **防御技能** | 格挡/护盾/反伤 | 0-2费 |
| **辅助技能** | Buff/Debuff | 0-2费 |
| **终结技** | 改变战局 | 3-4费，有冷却 |

**技能联动**：同一装备连续释放技能可触发"连携"（动能消耗）。
- 例如：先放"火焰斩"再放"烈焰风暴"，触发额外燃烧伤害。

### 5.4 能量机制（简化）

```
【能量系统】

基础能量（固定）：
├── 每轮ATB满时恢复3点
├── 能量上限5点
└── 慢速Build也能使用高费技能

动能（速度奖励）：
├── 速度>100时，每点速度+0.5%能量获取
└── 用于触发技能连携
```

## 六、天赋系统（装备的辅助强化）

### 6.1 天赋定位重构

**天赋是装备的"放大器"**，不是独立Build分支。

```
天赋功能：
├── 强化已有装备效果
├── 解锁装备隐藏词缀
└── 触发装备联动
```

### 6.2 天赋获取（简化）

| 层级 | 获取方式 | 重置 |
|------|---------|------|
| **基础天赋** | 每章自动解锁1个 | 休息点可重置 |
| **进阶天赋** | 击败精英/BOSS后三选一 | 不可重置 |

### 6.3 天赋树规模（修订为45节点）

| 类型 | 节点数 | 定位 |
|------|--------|------|
| 修真路线 | 15个 | 爆发/透支类强化 |
| 科技路线 | 15个 | 稳定/控制类强化 |
| 通用节点 | 15个 | 双路线通用强化 |

**天赋与装备联动**（专家建议）：
- 天赋与装备套装绑定：如"雷劫套装精通"仅在装备2件以上雷劫套装时解锁
- 天赋可解锁装备隐藏词缀（如"聚能"天赋解锁装备"能量上限+1"隐藏词缀）
- 部分天赋需要特定装备类型（如"武器专精·剑"）

### 6.4 天赋效果示例

| 天赋 | 效果 | 强化哪类装备 |
|------|------|------------|
| 聚能 | 能量上限+1 | 高费技能Build |
| 急攻 | ATB速度+15% | 快节奏Build |
| 元素亲和·火 | 火系伤害+20% | 元素Build |
| 套装精通 | 套装效果+25% | 套装流 |

## 七、新手引导设计（修订版）

### 7.1 第一局：装备体验局

- 给玩家一套**基础白装**（武器+护甲+2饰品+3个基础灵石）
- 引导理解"装备=技能来源"
- **增加ATB教学关**：强制玩家体验子弹时间
- 敌人中等难度，让玩家体验ATB博弈感
- 结局：解锁基础装备池

### 7.2 第二局：词缀+灵石教学局

- 开始掉落绿色/蓝色装备
- 引入词缀概念："这件装备多了个词缀，效果是..."
- **增加灵石槽教学**：介绍不同元素灵石的效果
- 引导激活第一个套装效果
- 结局：解锁锻造系统

**引导增强**（专家建议）：
- 第一局增加"词缀对比"：同时展示有/无词缀的装备战斗差异
- 增加"装备图鉴"：记录已见过的装备和词缀

### 7.3 死亡体验

```
死亡 → 展示"本局关键时刻"（不超过3个）
     → 获得"星尘"（局外成长，非碾压数值）
     → 获得"装备碎片"（集齐合成特定装备）
     → 下局可"刻印"1件装备（仅本局有效）
```

**刻印装备限制**（专家建议）：
- 只能刻印**稀有度≤蓝色**的装备
- 刻印时词缀**随机重置**（避免玩家刷开局定向词缀）
- 或改为"刻印装备的基础属性"，而非词缀

**星尘限制**：局外成长最高+15%，避免碾压

## 九、风险提示（修订版）

| 风险点 | 描述 | 应对策略 |
|--------|------|----------|
| ATB暂停机制 | 原"全局暂停"被专家否定 | 已改为子弹时间0.1x+行动队列 |
| 装备Build失衡 | ATB词缀可能主导Build | 同类词缀软上限3层+互斥机制 |
| 装备过于随机 | 玩家无法Build规划 | 保底机制+商店定向刷新+套装碎片 |
| 元素反应过载 | 15种组合认知成本高 | 分阶段解锁+UI反应预览提示 |
| 内容规模过大 | 100装备+80技能工作量巨大 | 已压缩至60件+45技能 |
| 刻印装备过强 | 可能导致刷开局定向 | 限制蓝色以下+词缀随机重置 |
| 套装凑齐困难 | 玩家长期无法集齐 | 套装碎片系统+2/3/4件套效果递减 |
| 技术架构脱节 | GDD与开发方案不一致 | 需同步更新开发方案v4.0 |

## 十、核心决策点（需用户确认）

1. **游戏定位**：采用**装备驱动型**（方案A），类似《侥幸骑士》
2. **ATB暂停机制**：采用"子弹时间(0.1x)"而非完全暂停
3. **能量恢复**：采用固定3点/轮，与速度脱钩
4. **元素反应**：全部15种，分三阶段解锁
5. **天赋定位**：天赋是装备的辅助强化，45节点
6. **词缀平衡**：同类词缀软上限3层，超出转动能
7. **刻印限制**：只能刻印蓝色以下装备，词缀随机重置
8. **开发方案同步**：需更新开发方案v4.0以匹配
