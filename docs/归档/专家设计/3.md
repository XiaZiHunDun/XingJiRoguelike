# 星陨纪元 · 游戏设计文档

## 一、设计理念

### 核心概念：**「瞬息抉择，万法归一」**

**想表达什么？**
- **时间压迫下的策略美学**：ATB蓄力条持续前进，玩家必须在"等待完美时机"和"抢先出手"之间做抉择——这是回合制无法提供的紧张感
- **科幻修真的融合不是皮肤，是机制**：修真=突破规则（元素爆发、越级伤害），科技=精确控制（稳定输出、资源管理），两条路线在玩法层面截然不同
- **Build的「涌现感」**：不是简单的数值叠加，而是元素+卡牌+天赋+装备四系统交叉产生的化学反应

---

## 二、系统设计详解

### 2.1 ATB半即时战斗 —— 差异化核心

**与回合制的本质差异：**

| 维度 | 传统回合制 | 本作的ATB半即时 |
|------|-----------|----------------|
| 决策压力 | 无时间压力，可无限思考 | 蓄力条持续前进，决策窗口有限 |
| 节奏控制 | 固定顺序 | 速度属性实时改变出手频率 |
| 打断机制 | 无或极少 | 核心玩法：蓄力中断、时机抢夺 |
| 资源博弈 | 静态规划 | 动态权衡：现在出牌 vs 等待更好时机 |

**放大差异的设计：**

```
【蓄力阶段】（0-70% ATB）
- 卡牌选择界面保持开启
- 玩家可随时确认出牌，但威力-20%（仓促惩罚）
- 敌人也在蓄力，可见其进度条

【临界点】（70-100% ATB）
- 卡牌威力正常
- 特殊机制：在90%时出牌触发「完美时机」，伤害+15%

【溢出风险】（100% ATB后）
- 每超10%损失5%最大生命值（灵气反噬）
- 强制自动打出当前选中卡牌（若无则随机）
```

**暂停机制的智慧：**
- 非全暂停，而是**子弹时间**（0.3倍速）
- 每次暂停最多3秒，冷却8秒
- 精英/BOSS战有「时停领域」可强制暂停（世界观：科技/修真的时间操控）

---

### 2.2 卡牌Build系统 —— 四维策略空间

**卡牌分类（不是简单标签，是机制差异）：**

| 类型 | 核心机制 | 能量特征 |
|-----|---------|---------|
| **攻击牌** | 可蓄力（按住增加威力，但消耗更多ATB） | 1-3费 |
| **防御牌** | 即时/持续/反伤三种子类型 | 1-2费 |
| **辅助牌** | 增益有「时效」：X次行动或X秒 | 0-2费 |
| **特殊牌** | 改变规则：重置ATB、偷取敌人进度、强制暂停 | 3-4费，有冷却 |

**能量机制创新：**
```
能量槽：5点上限，每20% ATB恢复1点
高费牌（3-4费）不是强在数值，而是强在「改变规则」
低费牌（0-1费）用于填充、触发连携、维持节奏

关键设计：「能量溢出」——满能量时继续恢复会转化为临时护盾
这避免了"憋大招"的最优解，鼓励能量流转
```

**Build深度来源：**
1. **元素附着**：每张牌带元素，敌人有「元素积蓄槽」，满槽触发异常
2. **卡牌连携**：同元素连续出牌有加成，不同元素触发反应
3. **速度协同**：快ATB多抽牌流 vs 慢ATB高威力流

---

### 2.3 元素系统 —— 化学反应而非数字克制

**六元素设计：**

| 元素 | 核心特性 | 异常状态 | 与ATB的关联 |
|-----|---------|---------|-----------|
| 火 | 爆发、持续伤害 | 燃烧：ATB恢复-30% | 高威力但可能打乱自身节奏 |
| 冰 | 控制、减速 | 冰冻：ATB暂停2秒 | 直接操控时间资源 |
| 雷 | 连锁、破防 | 感电：下次受伤+50% | 创造输出窗口 |
| 风 | 速度、连击 | 风缠：抽牌+1，持续10秒 | 资源获取 |
| 土 | 防御、反震 | 石化：受伤时反弹20% | 防御性节奏 |
| 物理 | 破盾、真实伤害 | 破甲：防御-50% | 稳定输出 |

**元素反应（双元素触发）：**
```
火+风 = 扩散：异常状态传播给周围敌人
火+冰 = 融化：伤害翻倍，但清除异常
雷+水（环境）= 感电领域：范围持续伤害
风+土 = 沙暴：视野受限，但闪避+30%
```

**与卡牌/天赋的协同：**
- 天赋「修真·焚天」：火元素异常可叠加3层，第3层瞬间爆发（改变火系玩法）
- 天赋「科技·冷凝」：冰元素异常精确控制，可手动选择冻结时机（科技的控制哲学）

---

### 2.4 天赋双路线 —— 真正的玩法分歧

**设计原则：不是「数值差异」，是「决策逻辑差异」**

| 维度 | 修真路线 | 科技路线 |
|-----|---------|---------|
| **核心哲学** | 突破极限，爆发至上 | 精确控制，稳定最优 |
| **ATB互动** | 可透支ATB（超100%）换取威力，但承受反噬 | ATB分段精确控制，每段有增益 |
| **元素玩法** | 元素异常叠加、爆发、无视抗性 | 元素转换、稳定附着、环境利用 |
| **资源管理** | 高风险高回报，能量可透支 | 能量循环，溢出利用 |
| **容错率** | 低，但上限极高 | 高，下限稳定 |

**避免「最优解」的设计：**

1. **环境适应性**：某些敌人对「爆发」有反制（伤害吸收转治疗），某些敌人对「稳定」有反制（随时间强化）
2. **混合奖励**：投入点数在双路线可解锁「星陨」特殊天赋（科幻修真融合）
3. **随机获取**：天赋不是自由点选，是战斗/事件掉落，玩家根据掉落构建

**天赋树规模建议：**
- 修真主干：15个节点（5层×3分支）
- 科技主干：15个节点
- 星陨融合：8个节点（需双路线各投入至少5点解锁）
- 总计：38个天赋节点

---

### 2.5 装备词缀 —— 构建的「调味剂」

**词缀设计原则：改变机制 > 改变数值**

| 类型 | 示例 | 设计意图 |
|-----|------|---------|
| ATB类 | 「时砂」：ATB上限+20%，但溢出惩罚翻倍 | 改变节奏风险 |
| 元素类 | 「元素共鸣」：元素反应范围+50% | 强化AOE构建 |
| 卡牌类 | 「循环」：打出3张同元素牌后，第4张0费 | 鼓励纯色Build |
| 特殊类 | 「逆命」：死亡时ATB重置为0并满血，每场战斗1次 | 翻盘机制 |

**词缀池规模：**
- T1普通词缀：20个（纯数值+小机制）
- T2稀有词缀：15个（显著改变玩法）
- T3传说词缀：10个（定义Build核心）
- 特殊套装词缀：5套×3件 = 15个（成套触发）

## 三、数值框架

### 3.1 ATB速度曲线

```
【基础速度】
- 玩家：100点/秒（满100出手）
- 普通敌人：80-120点/秒（差异化）
- 精英：100-140点/秒
- BOSS：有阶段变化，P1:90, P2:110, P3:130

【速度属性收益】
速度100→150：出手频率+50%（线性收益，但考虑能量恢复同步）
速度150→200：出手频率+33%（边际递减，防止无限快）

【决策窗口】
- 从选牌到必须出牌：平均4-6秒（70%→100% ATB）
- 子弹时间暂停：3秒（足够换牌，不够无限思考）
- 精英/BOSS战平均时长：90-150秒（3-5轮循环）
```

### 3.2 能量与卡牌费用

```
【能量循环】
- 基础恢复：每20% ATB = 1能量（即每轮出手恢复5能量）
- 卡牌平均费用：1.5能量
- 每轮净消耗：约3-4能量（5张牌）
- 能量上限：5（限制爆发，鼓励流转）

【费用分布】
- 0费：10%（填充、触发）
- 1费：40%（主力）
- 2费：35%（中坚）
- 3费：12%（改变局势）
- 4费：3%（终极手段）

【高费牌设计】
不是数值高，是机制强：
- 4费「时停」：强制暂停全场ATB 5秒
- 4费「万法归一」：本局使用过的所有元素同时附着
```

### 3.3 伤害公式

```
基础伤害 = 卡牌威力 × (1 + 元素加成 + 天赋加成) × 时机系数

时机系数：
- 仓促（<70% ATB）：0.8
- 正常：1.0
- 完美（90% ATB）：1.15

防御计算：
伤害减免 = 防御 / (防御 + 100)  （软上限，避免无敌）
```

## 五、用户体验设计

### 5.1 新手引导 —— 「三局教学法」

**第一局：教学局（不可失败）**
- 固定牌组：3攻击+2防御
- 强制暂停提示关键时机
- 敌人弱化，展示「完美时机」收益
- 结局：解锁天赋系统

**第二局：探索局**
- 开放卡牌选择（20张池子）
- 引入元素反应（火+冰教学）
- 首次遇到精英，展示天赋价值
- 结局：解锁装备系统

**第三局：完整局**
- 完整系统开放
- 首次遇到BOSS
- 死亡后展示「局外成长」

### 5.2 死亡与局外成长 —— 「星陨传承」

**挫败感消解设计：**
1. **死亡有产出**：每局收集「星尘」用于解锁新内容（不是数值成长，是新机制）
2. **死亡有故事**：根据死法生成「遗言」，记录本局高光时刻
3. **死亡有选择**：下局可选择继承一个「执念」（上局某张牌或天赋，首局免费）

**局外成长（非数值）：**
```
解锁内容：
- 新卡牌加入池子（不改变旧卡强度）
- 新天赋节点
- 新敌人（更多挑战，不是更强）
- 新事件剧情
- 皮肤/特效（炫耀性）

不解锁：
- 基础属性成长（避免碾压）
- 开局优势（保持每局新鲜感）
```

## 六、风险提示

### 6.1 高风险设计

| 风险点 | 说明 | 缓解方案 |
|-------|------|---------|
| **ATB节奏失控** | 速度堆叠过快导致战斗混乱 | 设置速度软上限200，超过后转收益 |
| **暂停滥用** | 玩家频繁暂停失去紧张感 | 暂停有冷却，BOSS有反暂停机制 |
| **元素反应平衡** | 某些反应过强成为唯一解 | 敌人有「元素抗体」机制，强制多样性 |
| **Build同质化** | 社区快速找出最优解 | 每周轮换「星陨变异」（全局规则变化） |
| **科幻修真割裂** | 世界观停留在皮肤 | 确保两条路线玩法差异体现哲学差异 |

### 6.2 开发优先级

## 七、核心差异放大总结

**ATB vs 回合制的「杀手锏」：**

1. **可见的压迫感**：敌人蓄力条在你眼前涨，你知道它要出大招了，但你还没准备好——这种焦虑回合制给不了
2. **时机博弈**：不是「我回合做什么」，是「现在出牌值不值，还是再等等」——每次决策都有机会成本
3. **打断的艺术**：用控制技能打断敌人90%的蓄力，这种「抢时间」的爽感是核心乐趣
4. **节奏多样性**：快攻流（高频低伤）和蓄力流（低频高伤）在ATB下是截然不同的操作节奏，不是简单的数值差异

**科幻+修真的玩法体现：**

| 世界观概念 | 玩法映射 |
|-----------|---------|
| 灵气/科技冲突 | 天赋双路线的互斥与融合 |
| 突破境界 | 透支ATB的爆发机制（修真） |
| 精密仪器 | 能量循环与时机控制（科技） |
| 星陨事件 | 全局规则变化的Roguelike变异 |
| 古老遗迹 | 特殊事件获取一次性强力天赋 |

---

这套设计的核心是让**「时间」成为真正的资源**，玩家不是在管理卡牌，是在管理**时间压力下的决策质量**。科幻修真的融合不是背景故事，是**两种截然不同的时间管理哲学**——一个要打破规则（快、爆发、高风险），一个要优化规则（稳、控制、高效率）。
