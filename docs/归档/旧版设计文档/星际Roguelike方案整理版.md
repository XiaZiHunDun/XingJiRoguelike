## 第一章 游戏概述

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 游戏名称 | 星际Roguelike（星陨纪元） |
| 类型 | Roguelike + ATB半即时战斗 + 装备驱动Build |
| 平台 | Steam 单机 |
| 核心循环 | 探索 → 战斗 → 掉落装备 → Build构建 → 境界提升 → 死亡重来 |

### 1.2 世界设定

**背景故事**
```
玩家是一名坠毁在未知星球的星际旅者。
这个星球曾是繁荣的星际文明的家园，但文明已经衰落。
星球上有5个主要区域，每个区域都有独特的古代文明遗迹。
在遗迹中弥漫着"星尘粒子"，玩家通过吸收星尘觉醒修炼之路。
玩家需要在遗迹中探索、收集，建立Build，提升境界，最终找到离开星球的方法。
```

**阵营势力**
```
星球上存在多个修炼势力，各有不同的理念和资源。
玩家可以加入某个势力，获得势力独特的装备和任务。
玩家同时只能加入一个阵营。
守墓人是唯一敌对阵营，击杀可获得贡献。
```

### 1.3 核心体验

```
【传统Roguelike】
拿到什么用什么 → 被动接受随机性

【我们的方案】
看到随机性 → 做决策 → 构建Build → 享受化学反应
```

## 第二章 战斗系统

### 2.1 ATB半即时战斗

```
【ATB机制】
├─ ATB槽：每个角色和敌人有一个蓄力槽
├─ 蓄力速度 = 100 + 敏捷 × 3 + 装备加成
├─ 槽满即可行动（普攻/技能/道具/逃跑）
├─ 速度软上限200，超出转化伤害加成
├─ 子弹时间：槽蓄满前一刻触发0.1x慢动作
└─ 时砂系统：3次/局，2秒真暂停

【行动类型】
├─ 攻击：使用普攻或技能
├─ 技能：消耗能量释放强力技能
├─ 道具：使用消耗品
└─ 逃跑：放弃战斗（可能损失掉落）

【完美时机】
├─ 时机加成：完美(≥90%) × 1.15，仓促(<70%) × 0.8
└─ 子弹时间触发时，可以选择行动
```

### 2.2 属性伤害系统

```
【伤害类型】
└─ 物理/奥术

【伤害类型特性】
├─ 物理：受防御减免
├─ 奥术：受魔抗减免，能量涌动
└─ 速度溢出：ATB速度超过200时，超出部分转化为伤害加成

【属性联动】
├─ 物理联动：速度溢出高时，物理伤害+15%
└─ 奥术累积：秘法满层 → 技能冷却-20%
```

### 2.3 战斗数值

```
【受伤计算流程】
敌方攻击 → 闪避判定 → 伤害减免 → 防御减伤 → 最终伤害

最终伤害 = (原始伤害 × 伤害减免) - 防御力

【闪避相关】
闪避率 = min(50%, 敏捷 × 0.4% + 装备加成)

【暴击相关】
暴击率 = min(80%, 敏捷 × 0.5% + 装备加成)
暴击伤害 = 150% + 敏捷 × 0.2% + 装备加成

【ATB速度】
ATB速度 = 100 + 敏捷 × 3 + 装备加成
速度溢出：ATB速度超过200时，超出部分获得额外效果（伤害+1%每点溢出）
```

## 第四章 境界系统

### 4.1 境界等级

```
【境界划分】（Phase 1 仅开放至星火境）

凡人境（1-10级）
├─ 描述：初入修炼，吸收星尘
└─ 特征：基础属性成长

感应境（11-20级）
├─ 描述：感应星尘粒子
└─ 特征：解锁元素亲和

聚尘境（21-30级）
├─ 描述：星尘凝聚成型
└─ 特征：技能强度提升

凝核境（31-40级）
├─ 描述：星尘核蜕变为星尘之躯
└─ 特征：所有属性大幅提升

星火境（41-50级）
├─ 描述：星尘燃烧，化为星火
└─ 特征：所有属性达到顶峰

【后续境界】（Phase 2+ 启用）
└─ 脉冲境 → 裂变境 → 黑洞境 → 创世境（待宇宙星球地图开发后启用）
```

### 4.2 境界作用

```
1. 解锁穿戴要求：高境界装备需要对应境界
2. 解锁技能变体：每10级解锁一个变体
3. 属性成长：升级增加体质/精神/敏捷
```

## 第五章 装备系统

### 5.1 核心设计

```
【取消】
├─ 稀有度系统（凡品/稀有/史诗/传说）
└─ 装等系统（T1-T6）

【替代】穿戴要求系统
├─ 装备掉落时随机生成穿戴要求
├─ 玩家属性/技能/境界满足要求才能穿戴
└─ 高要求 = 通常意味着高收益
```

### 5.2 掉落随机性

```
【技能】（每件装备，各20%均匀分布）
├─ 0技能：20%（纯词缀装）
├─ 1技能：20%
├─ 2技能：20%
├─ 3技能：20%
└─ 4技能：20%

【词缀】
├─ 数量：1-4个
└─ 类型：从词缀池随机

【穿戴要求】（三维随机组合）
├─ 属性要求（体质/精神/敏捷）
├─ 境界要求（对应装备境界等级）
└─ 技能要求（特定技能数量）
```

### 5.3 装备类型

```
【武器】
├─ 巨剑：物理攻击为主
├─ 法杖：魔法攻击为主
└─ 双刃：物理暴击为主

【护甲】
├─ 钢甲：高防御，高体质
├─ 法袍：高魔法攻击，高精神
├─ 皮甲：高敏捷，高速度
└─ 修真袍：综合

【饰品】（2个槽位）
├─ 火魂戒、暴击戒、守护戒等
└─ 提供各种词缀效果

【增幅器】（3个槽位，特殊装备格子）
├─ 体质增幅器：体质×1.5/×2.0/×2.5（绿色科技品质）
├─ 精神增幅器：精神×1.5/×2.0/×2.5（蓝色科技品质）
└─ 速度增幅器：敏捷×1.5/×2.0，ATB速度+10%/+20%（紫色科技品质）

【增幅器说明】
├─ 增幅器为科技装备，不带技能和词缀
├─ 增幅器品质由低到高：绿色→蓝色→紫色
├─ 增幅器通过贡献兑换或任务奖励获得
└─ 增幅器不可出售，不可重铸
```

### 5.4 唯一性装备

```
【世界唯一装备】
├─ 整个世界只有一件（同一时间）
├─ 获取方式：任务奖励/成就/守墓人兑换/极低概率随机掉落
├─ 死亡后消失，可再次获取
└─ 6件独特装备
```

## 第九章 敌人系统

### 9.1 敌人分类

```
【普通敌人：原生异兽】
├─ 在文明灭亡后占据遗迹的本土生物
├─ 每区域有3-5种基础类型
└─ 示例：沙蝎、冰霜狼、机魂蜘蛛、翠藤蛇

【精英敌人：古代守卫】
├─ 文明灭亡后仍在运作的守护构造体
├─ 每区域有1-2种独特类型
└─ 示例：沙神守卫、冰魔像、机魂巨人

【BOSS：区域之主】
├─ 每区域1个，代表区域终极威胁
├─ 独特的阶段和机制
```

### 数值计算参考

```
【角色属性成长】（星际战士为例）
初始属性：体质40，精神30，敏捷30

每级成长：
├─ HP增量 = 体质 × 8
├─ 能量增量 = 精神 × 0.5
└─ ATB速度增量 = 敏捷 × 1.5

15级时（假设无装备加成）：
├─ HP = 100 + 40×8 + 14×8 = 532
├─ 能量 = 5 + 30×0.5 = 20
└─ ATB速度 = 100 + 30×3 = 190
```
