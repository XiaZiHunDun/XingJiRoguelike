# 《星陨纪元》核心系统设计文档

## 一、设计理念

### 1.1 游戏核心愿景

> **"以命为棋，落子无悔" —— 在时间与命运的双重博弈中，书写属于你的修真传奇。**

游戏融合科幻修真世界观：修仙文明与星际科技交织，古老法则与量子理论共鸣。玩家扮演星陨纪元中的"星陨行者"，在ATB半即时战斗中，通过卡牌构筑与天赋觉醒，对抗命运、突破天道。

### 1.2 核心理念表达

| 理念维度 | 表达方式 | 玩家感知 |
|---------|---------|---------|
| **时间即资源** | ATB槽决定行动顺序，暂停机制创造策略空间 | "我必须快，但不能盲目快" |
| **Build即身份** | 卡牌+天赋+装备组合决定玩法流派 | "我的Build独一无二" |
| **命运可改写** | Roguelike单局+局外成长交织 | "每一次死亡都是进化" |
| **科幻×修真** | 修真丹田=能量核心，星舰科技=辅助系统 | "古老与未来在此合一" |

### 1.3 设计哲学：避免"不上不下"的ATB陷阱

传统ATB的尴尬在于——**不够即时（缺乏紧张感）又不完全是回合（策略可以慢慢想）**。我们的解法是：

```
┌─────────────────────────────────────────────────────────────┐
│                    ATB设计三角平衡                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│           ▲ 即时感                                          │
│          /│\                                                │
│         / │ \        高速角色：高频决策                      │
│        /  │  \       → 策略压力大，奖励刺激感                │
│       /   │   \                                              │
│      /────┴────\     慢速角色：充分准备                      │
│     /    低速     \    → 决策从容，享受Build成果             │
│    /                \                                        │
│   ──────────────────── 决策深度                             │
│                                                             │
│   核心设计：速度差异 + 暂停机制 + 时间膨胀/收缩               │
└─────────────────────────────────────────────────────────────┘
```

**差异化策略**：
- **高速角色（敏捷/雷系）**：ATB增长快→决策窗口短→每局多次行动→刺激感强
- **低速角色（力量/土系）**：ATB增长慢→决策窗口长→单次行动影响大→策略感强
- **科技路线**：可主动"时间膨胀"延长友方ATB窗口，缩短敌方窗口
- **修真路线**：可"因果预视"临时加速自身，跳过等待期

## 二、系统设计

### 2.1 ATB半即时战斗系统

#### 2.1.1 基础机制

```
ATB槽（0-100）
     │
     │  速度决定蓄力速率（点/秒）
     │
     ▼
ATB满 → 进入"决策态"（时间暂停） → 选择卡牌 → 执行 → ATB清零重新蓄力
```

**关键数值设计**：

| 参数 | 数值 | 设计理由 |
|-----|------|---------|
| ATB上限 | 100 | 便于计算，视觉清晰 |
| 基础速度 | 10点/秒 | 满ATB需10秒，符合"半即时"节奏 |
| 决策超时 | 15秒 | 超时自动普攻，避免拖延 |
| 时间膨胀系数 | 0.3x / 2x | 科技/修真路线核心差异 |
| 最大等待惩罚 | 80%速度衰减 | 防止无脑等待 |

#### 2.1.2 ATB速度曲线优化

```
速度档位（1-10级）:
┌──────────────────────────────────────────────────────────────┐
│ 10级(极速) ─────── 25点/秒 ────── 4秒满槽 ── 高频决策，高刺激  │
│  7级(快速) ───────── 18点/秒 ───── 5.5秒满槽 ── 平衡点        │
│  5级(标准) ───────── 10点/秒 ───── 10秒满槽 ── 新手舒适区     │
│  3级(缓慢) ───────── 6点/秒 ────── 16秒满槽 ── 策略深度      │
│  1级(龟速) ───────── 3点/秒 ────── 33秒满槽 ── 特殊Build     │
└──────────────────────────────────────────────────────────────┘

速度增长曲线：非线性递减
每提升1级速度，实际增长≈前一级差的70%
→ 防止极端Build统治游戏
```

#### 2.1.3 决策窗口设计

**决策窗口时长 = 基础窗口 × 角色速度系数**

| 角色速度 | 决策窗口 | 预期行为 |
|---------|---------|---------|
| 10级 | 2-3秒 | 肌肉记忆+预判，几乎无法深思 |
| 7级 | 4-5秒 | 快速浏览+决策，仍有紧张感 |
| 5级 | 6-8秒 | 正常决策时间，舒适区 |
| 3级 | 10-12秒 | 充分思考，可做复杂计算 |
| 1级 | 15-18秒 | 充裕时间，但可能无聊 |

**优化方案**：
- **分层暂停**：低威胁敌人自动延迟2秒，给玩家"喘息"
- **威胁预警**：高伤害技能前2秒闪烁提示
- **决策回放**：结束后可复盘决策窗口内的操作

#### 2.1.4 ATB vs 回合制的核心差异

| 维度 | 回合制 | 纯即时 | 我们的ATB |
|-----|-------|-------|----------|
| 时间感知 | 无 | 持续流逝 | **动态感知** |
| 策略时间 | 无限 | 极度压缩 | **速度决定** |
| 战斗节奏 | 可暂停 | 不可控 | **可干预** |
| 角色差异 | 回合顺序固定 | 难以差异化 | **速度即特色** |
| 核心乐趣 | 策略深度 | 反应速度 | **时间博弈** |

**放大差异的机制**：
1. **速度压制**：高速角色可"插队"打断低速角色蓄力
2. **时间裂隙**：特定卡牌消耗"时间碎片"实现额外行动
3. **因果律技能**：可改变己方/敌方ATB流动速率
4. **先手优势递减**：连续先手，下一轮速度降低（防止滚雪球）


#### 2.2.2 卡牌设计原则

**原则1：每张卡牌必须有"决策点"**
- ❌ 错误：火球术——造成10点伤害
- ✅ 正确：火球术——造成8点伤害，选择目标。若目标已有灼烧状态，则额外眩晕1回合

**原则2：避免纯数值碾压**
- 高费卡 ≠ 高伤害
- 4费卡可能只是"移除敌方一个增益状态"
- 价值在于"解决特定问题的能力"

**原则3：卡牌联动创造涌现性**
- 单卡平庸 → 组合爆炸
- 示例：冰系减速(1费) + 雷系连锁(2费) = 减速后连锁伤害+150%

**原则4：语义明确的视觉效果**
- 每张卡牌有唯一主色调（元素色）
- 动画时长与能量消耗正相关
- 高费卡播放专属特写动画

#### 2.2.3 卡牌获取与构筑策略

**Roguelike卡牌获取节奏**：

| 阶段 | 节点数 | 预期卡牌 | 策略重点 |
|-----|-------|---------|---------|
| 开局(第1-3章) | 3-5张 | 过渡牌为主 | 建立手感，测试元素偏好 |
| 中期(第4-7章) | 8-12张 | 核心牌出现 | 明确Build方向，收割联动牌 |
| 后期(第8-章) | 15-20张 | 关键补强 | 查漏补缺，针对BOSS调整 |

**单局牌组规模目标**：
- 理想牌组：18-25张
- 最优曲线：2/3核心牌 + 1/3灵活应对牌
- 避免"抽不到"：确保关键牌有多种获取途径

#### 2.2.4 卡牌与元素系统协同

元素系统不是独立存在的——它必须与卡牌Build深度耦合：

1. **灵识系**：强化灵识判定，提升暴击率和暴击伤害
   - 核心天赋：因果预视（可预览敌人下一行动）
2. **丹道系**：能量管理，灵力恢复与透支
   - 核心天赋：轮回丹（死亡时复活一次，消耗50%灵力）
3. **渡劫系**：元素共鸣，强化元素反应
   - 核心天赋：天雷淬体（雷系伤害转化为真实伤害）

**标志性机制**：
- **因果预视**：战斗中可查看敌人即将释放的技能
- **灵力透支**：暂时突破能量上限，下回合强制进入"枯竭"状态
- **渡劫反击**：受到致命伤害时，有概率触发天劫反击

#### 2.4.4 科技路线详解

**三大分支**：
1. **计算系**：精准控制ATB与技能冷却
   - 核心天赋：时间膨胀场（指定友方ATB速度×2，持续5秒）
2. **护盾系**：护盾值管理与反击机制
   - 核心天赋：纳米重构（护盾被击破时转化为生命值）
3. **增幅系**：能量效率与多目标打击
   - 核心天赋：电磁脉冲（沉默范围内敌人，禁用技能2回合）

**标志性机制**：
- **时间膨胀/收缩**：主动干预战场ATB节奏
- **护盾重构**：护盾不被消耗，而是"延迟生效"
- **精准计算**：概率触发额外行动，但不触发敌人反击

#### 2.4.5 避免"最优解"的策略

**策略1：敌人适应性设计**
- 每章节BOSS有"反制机制"——需要特定路线才能轻松应对
- 示例：某BOSS免疫控制 → 纯控制Build效率低
- 示例：某BOSS高护甲 → 纯物理Build需要元素穿透

**策略2：天赋共鸣强制混搭**
- 核心天赋需要"前置条件"——必须点亮对方路线的某些节点
- 例：修真顶级天赋"天人合一"需要先点科技系的"基础能量管理"

**策略3：动态难度调整**
- 当某种Build胜率超过60%，自动提升对应敌人强度
- 反之亦然——弱势Build会获得额外加成

**策略4：成就/里程碑驱动**
- 不只是"通关"，还有"用X流派通关"的成就
- 不同流派解锁不同隐藏内容，鼓励多样性尝试

- **可见性**：玩家能看到下1-2步的节点，增加规划感
- **分支性**：每章至少2条路径可选，避免一条路走到黑
- **节奏感**：战斗密集区后必有喘息点（商店/休息）
- **隐藏内容**：特殊节点需特定条件触发（不同Build体验不同）

```

实际伤害 = 基础伤害 × (1 + 增伤%) × 克制修正 × (1 - 减伤%) × 暴击修正

详细参数：
┌─────────────────────────────────────────────────────────────┐
│ 基础伤害：卡牌固定值 + 角色属性加成                            │
│ 增伤来源：装备词缀、天赋被动、元素亲和、特殊状态                │
│ 克制修正：克制=1.25 / 被克制=0.85 / 无克制=1.0               │
│ 减伤来源：护甲（线性递减）、护盾（优先消耗）、闪避（概率免伤）   │
│ 暴击修正：暴击伤害 = 基础伤害 × 2.0（可被天赋提升至3.0）       │
└─────────────────────────────────────────────────────────────┘

元素反应伤害：
┌─────────────────────────────────────────────────────────────┐
│ 二阶反应基础伤害：角色等级 × 8                                │
│ 三阶反应基础伤害：角色等级 × 15                               │
│ 反应伤害享受：增伤加成、元素亲和加成                            │
│ 反应伤害不享受：暴击加成、克制加成（已是额外独立伤害）           │
└─────────────────────────────────────────────────────────────┘
```

## 七、风险提示

### 7.1 开发阶段风险

| 风险点 | 严重性 | 应对策略 |
|-------|-------|---------|
| **ATB平衡难以把控** | 高 | 优先开发速度档位数值表，预留动态调整接口 |
| **元素反应组合爆炸** | 高 | 从二阶反应开始验证，三阶反应后置；建立反应效果白名单 |
| **卡牌数量不足** | 中 | 先保证核心体验（每元素5-8张），后续扩展；避免早期过度设计 |
| **天赋树深度不足** | 中 | 80%节点为"可选强化"，20%为核心质变节点；后期可增加新分支 |
| **装备词缀同质化** | 中 | 词缀设计分层：基础填充→Build强化→质变专属；建立词缀价值评估表 |

### 7.2 平衡性风险

| 风险点 | 描述 | 应对策略 |
|-------|------|---------|
| **最优解问题** | 某Build明显强于其他 | 建立Build强度评估模型；动态难度调整；定期平衡补丁 |
| **克制溢出** | 元素克制过于强力 | 克制伤害上限；提供克制抵抗词缀；BOSS设计避免单一元素 |
| **能量两极化** | 0费卡滥用或高费卡无人用 | 0费卡使用次数限制；高费卡增加"溢出能量"奖励 |
| **天赋单路线碾压** | 专精一路优于双线发展 | 跨界融合节点强制混搭；BOSS设计针对单一路线 |
| **ATB速度极端Build** | 10速角色vs1速角色体验割裂 | 速度衰减机制（连续先手惩罚）；战斗外速度不影响数值 |

### 7.3 内容深度风险

| 风险点 | 描述 | 应对策略 |
|-------|------|---------|
| **重复游玩动力不足** | 第一次通关后缺乏动力 | 多结局、隐藏内容、成就系统；Roguelike+元素保证每次体验不同 |
| **内容消耗速度** | 玩家比预期更快消耗内容 | 规划好Early Access内容节奏；社区反馈驱动后续内容 |
| **卡牌深度不足** | 卡牌差异性小，策略有限 | 卡牌设计必须包含"决策点"；建立卡牌评价体系（控场/爆发/过牌等） |

## 九、科幻×修真世界观融合设计

### 9.1 融合理念

```
"修真，是宇宙熵减的终极追求；科技，是文明逆熵的智慧结晶。"

在星陨纪元，古老修仙者驾驶星舰穿越星系，
星际文明研究"灵气"作为新型能源。
两者的碰撞，不是简单叠加，而是有机融合。
```

### 9.2 玩法层面的融合体现

| 融合维度 | 修真体现 | 科技体现 | 融合效果 |
|---------|---------|---------|---------|
| **能量系统** | 灵力（波动、透支、爆发） | 能量（稳定、可控、计算） | 双轨并行，可互相转化 |
| **ATB干预** | 因果预视、灵识加速 | 时间膨胀场、时空锁定 | 修真改变"因"，科技改变"果" |
| **元素技能** | 真火炼魂、天雷渡劫 | 等离子武器、量子冻结 | 外形科幻，效果修真 |
| **装备设计** | 法器飞剑、丹药护甲 | 纳米装甲、能量护盾 | 修真魂+科技皮 |
| **天赋路线** | 灵识/丹道/渡劫 | 计算/护盾/增幅 | 各有特色，跨界融合 |
| **世界观叙事** | 天道因果、轮回因果 | 星际探索、文明演进 | 同一个宇宙的不同视角 |

## 十、总结

### 10.1 核心理念回顾

> **"时间即资源，Build即身份，命运可改写"**

《星陨纪元》的核心乐趣在于：
1. **ATB时间博弈**：速度差异创造独特的决策压力与刺激感
2. **卡牌Build深度**：卡牌×元素×天赋的涌现性组合
3. **科幻修真融合**：古老法则与星际科技的碰撞，不是噱头而是机制

### 10.2 关键成功指标

| 维度 | 目标 | 验证方法 |
|-----|------|---------|
| 核心乐趣 | ATB决策紧张感 | 玩家反馈：是否"紧张又享受" |
| Build多样性 | 5种以上可行流派 | 数据：通关Build类型分布 |
| 重玩价值 | 单次体验>3小时 | 数据：首次通关后的重开率 |
| 融合效果 | 玩家感知"独特" | 反馈：是否感觉"这游戏不一样" |

### 10.3 下一步行动建议

```
□ 第一周：完成ATB核心循环的纸上原型（数值表+流程图）
□ 第二周：制作ATB战斗Demo，验证"暂停→决策→执行"流程
□ 第一个月：完成MVP（1章节+10张核心卡牌+基础敌人）
□ 第三个月：完成第二阶段（元素系统+天赋树+完整章节）
□ 第六个月：Steam EA发布
```
