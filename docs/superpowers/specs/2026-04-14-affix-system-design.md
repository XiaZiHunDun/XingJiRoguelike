## 1. 词缀分类概览

### 1.1 词缀分布

| 类型 | 占比 | 数量 | 特点 |
|------|------|------|------|
| 恒定型 | 30% | 16个 | 持续生效的数值加成（含属性/伤害/战斗属性） |
| 触发型 | 20% | 8个 | 满足条件时触发特殊效果 |
| 代价型 | 10% | 8个 | 高风险高回报 |
| 形态改变型 | 15% | 8个 | 改变技能形态/行为 |
| 魔法增强型 | 25% | 5个 | 强化魔法伤害和效果 |

### 1.2 词缀池设计

```
【词缀池机制】
├─ 每件装备随机从对应类型池中抽取
├─ 词缀效果独立，不受其他装备影响
├─ 同类型词缀不重复出现（同件装备）
└─ 不同装备的同类型词缀可叠加
```

### 触发型词缀详细说明

```
【斩杀追击】
├─ 触发：击杀任意敌人
├─ 效果：ATB+20%
└─ 用途：加速循环，击杀后快速下一轮

【低血狂暴】
├─ 触发：HP<30%时
├─ 效果：伤害+30%
└─ 用途：残血反杀条件

【完美时机】
├─ 触发：ATB槽100%时行动
├─ 效果：伤害+20%
└─ 用途：鼓励精准时机操作

【速度爆发】
├─ 触发：ATB速度>200时
├─ 效果：伤害额外+50%
├─ 联动：速度溢出机制
└─ 示例：ATB速度250 → 伤害+50%(溢出) + 50%(词缀) = +100%

【连锁奥术】
├─ 触发：击杀敌人时
├─ 效果：奥术链攻击最多3个附近敌人
├─ 伤害：魔法攻击×50%
└─ 用途：AOE清场

【以牙还牙】
├─ 触发：成功格挡伤害时
├─ 效果：反击敌人，造成物理攻击×50%伤害
├─ 额外：敌人ATB倒退10%
└─ 用途：反击流Build

【暴击回能】
├─ 触发：暴击时
├─ 效果：恢复1点能量
├─ 联动：与暴击率Build配合
└─ 用途：能量循环加速

【护盾反弹】
├─ 触发：护盾被打破时
├─ 效果：对周围敌人造成护盾值×50%伤害
└─ 用途：防守反击
```

```
【横斩·弧光】
├─ 基础效果：对单体造成100%物理攻击伤害
├─ 改变后：180°扇形，范围+50%
├─ 用途：同时攻击多个敌人
└─ 视觉：刀光变为弧形斩击

【横斩·穿刺】
├─ 基础效果：对单体造成100%物理攻击伤害
├─ 改变后：穿透敌人，无视40%防御
├─ 用途：对付高防御敌人
└─ 视觉：刀光变为穿刺光线

【流星·分裂】
├─ 基础效果：对范围内敌人造成200%伤害
├─ 改变后：分裂为3颗小流星，随机落点
├─ 总伤害：不变（3×67%≈200%）
├─ 用途：覆盖更大范围
└─ 视觉：1个大流星变为3个小流星

【铁壁·荆棘】
├─ 基础效果：获得护盾值=防御×3
├─ 改变后：护盾存在时，格挡伤害反弹30%
├─ 用途：防守反击
└─ 视觉：护盾表面有荆棘纹理

【奥术弹·能量倾泻】
├─ 基础效果：对单体造成120%魔法攻击伤害
├─ 改变后：单体3次伤害，每次40%（总伤害×1.5）
├─ 用途：单体爆发
└─ 视觉：3道连续奥术冲击

【闪现·幻影】
├─ 基础效果：短距离瞬移，获得1秒无敌帧
├─ 改变后：留下分身吸引敌人攻击2秒
├─ 用途：嘲讽聚怪
└─ 视觉：分身外观与角色相同

【法术护盾·寒霜】
├─ 基础效果：获得护盾值=魔法攻击×2
├─ 改变后：护盾破碎时冻结周围敌人2秒
├─ 用途：防守控制
└─ 视觉：护盾为冰蓝色，破碎时产生冰爆

【奥术风暴·连锁】
├─ 基础效果：对范围内敌人造成300%伤害
├─ 改变后：闪电在敌人间弹射3次，每次50%
├─ 总伤害：300% + 3×50% = 450%（理想状态）
├─ 用途：对付分散敌人
└─ 视觉：风暴中产生闪电链
```

### 形态改变规则

```
【叠加规则】
├─ 每件装备最多1个形态改变词缀
├─ 不同装备的形态改变词缀可以叠加
├─ 例：武器有"横斩·弧光"，饰品有"横斩·穿刺" → 横斩同时有弧光和穿刺效果
└─ 效果叠加遵循：范围扩大+无视防御可以并存
```

## 7. 词缀共鸣

### 7.1 基础共鸣（2个同系词缀触发）

| 共鸣名称 | 触发条件 | 效果 |
|----------|----------|------|
| 物理共振 | 2个物理伤害词缀 | 物理伤害+5% |
| 奥术共振 | 2个魔法伤害词缀 | 奥术伤害+5% |
| 暴击共振 | 2个暴击相关词缀 | 暴击率+3% |

### 7.2 进阶共鸣（3个同系词缀触发）

| 共鸣名称 | 触发条件 | 效果 |
|----------|----------|------|
| 物理精通 | 3个物理伤害词缀 | 物理伤害+15% |
| 奥术精通 | 3个魔法伤害词缀 | 魔法伤害+20%，技能冷却-10% |
| 暴击风暴 | 3个暴击相关词缀 | 暴击率>30%时，暴击后伤害×1.5 |

### 7.3 高级共鸣（4个同系词缀触发）

| 共鸣名称 | 触发条件 | 效果 |
|----------|----------|------|
| 物理极致 | 4个物理伤害词缀 | 物理伤害+30%，物理技能范围+20% |
| 奥术极致 | 4个魔法伤害词缀 | 奥术伤害+35%，能量消耗-20% |
| 速度爆发 | ATB速度>250 | 速度溢出伤害+30% |

### 共鸣详细说明

```
【共鸣计算示例】
场景：玩家有3个物理伤害词缀的散件

装备1：锋利（物理伤害+10%）
装备2：暴戾（暴击伤害+20%）→ 触发暴击共鸣
装备3：斩杀追击（触发型）→ 不计入物理词缀

当前共鸣：
├─ 物理伤害词缀：1个（不足2个，无共鸣）
├─ 暴击词缀：2个（满足暴击共振）→ 暴击率+3%
└─ 触发型词缀：1个（不计入基础共鸣）
```
