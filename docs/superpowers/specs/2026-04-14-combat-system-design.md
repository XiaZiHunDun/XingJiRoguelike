## 1. ATB机制详解

### 1.1 ATB槽结构

```
【ATB槽】
├─ 最大值：100%
├─ 初始值：0%
├─ 蓄力速度 = 100 + 敏捷 × 3 + 装备加成
├─ 每秒蓄力进度 = ATB速度 / 10（游戏中1秒 = 实际约1秒）
└─ 软上限：200，超出部分转化为伤害加成
```

**蓄力速度示例：**

| 敏捷 | 基础ATB速度 | 实际蓄力时间(满槽) |
|------|------------|------------------|
| 20 | 160 | 0.625秒 |
| 30 | 190 | 0.526秒 |
| 40 | 220 | 0.455秒 |
| 50 | 250 | 0.4秒（溢出50点） |

### 1.2 速度溢出机制

```
【速度溢出】
├─ 触发条件：ATB速度 > 200
├─ 溢出计算：伤害加成 = (ATB速度 - 200) × 1%
├─ 溢出加成类型：所有伤害（物理+奥术）
└─ 示例：ATB速度250 → 伤害+50%

【溢出联动】
├─ 物理系：ATB速度 > 250时，物理伤害额外+15%
└─ 奥术系：每累积3层秘法，能量消耗-5%
```

### 1.3 子弹时间

```
【子弹时间触发】
├─ 触发时机：ATB槽达到100%的前0.5秒
├─ 时间倍率：0.1x（正常1秒 = 子弹时间10秒）
├─ 持续时间：玩家选择行动后结束
└─ 视觉表现：屏幕边缘泛光，敌人动作放缓

【子弹时间内的选择】
├─ 可查看：所有敌人ATB进度、所有技能冷却
├─ 可执行：普攻/技能/道具/逃跑
└─ 完美时机：ATB槽刚好满时(100%)行动，伤害×1.15
```

### 1.4 时砂系统

```
【时砂】
├─ 数量：3次/局
├─ 效果：2秒真暂停（时间完全静止）
├─ 使用方式：快捷键（默认Z键）
├─ 用途：关键时刻思考策略使用
└─ 限制：不能在使用时砂期间进行任何操作
```

### 2.3 能量系统

```
【能量恢复】
├─ 初始能量：5点
├─ 最大能量：5 + 精神 × 0.5 + 装备加成
├─ 能量恢复方式：
│   ├─ 每回合结束：+0.5能量
│   ├─ 暴击时：+1能量（需装备"暴击回能"词缀）
│   └─ 击杀敌人：+0.5能量
└─ 能量上限示例（无装备）：
    精神30 → 能量上限20
    精神40 → 能量上限25
    精神50 → 能量上限30
```

## 3. 伤害计算

### 3.1 受伤计算流程

```
敌方攻击
    ↓
[1] 闪避判定（先判定是否闪避）
    ↓ 闪避成功 → 0伤害
    ↓ 闪避失败
[2] 伤害减免（百分比减伤）
    ↓
[3] 防御/魔抗减伤（固定值减伤）
    ↓
[4] 最终伤害
```

### 3.2 闪避判定

```
【闪避率计算】
闪避率 = min(50%, 敏捷 × 0.4% + 装备加成)

【闪避示例】
敏捷20 → 闪避率8%
敏捷30 → 闪避率12%
敏捷40 → 闪避率16%
敏捷50 → 闪避率20%
敏捷70 → 闪避率28%（接近上限）
敏捷100+ → 闪避率50%（达到上限）

【闪避规则】
├─ 闪避成功后，敌人ATB不消耗（该回合浪费了攻击）
├─ 某些技能必中（如奥术风暴范围攻击，不可闪避）
└─ BOSS攻击有最低20%命中率
```

### 3.3 暴击判定

```
【暴击率计算】
暴击率 = min(80%, 敏捷 × 0.5% + 装备加成)

【暴击伤害计算】
暴击伤害 = 150% × (1 + 敏捷 × 0.2% + 装备加成%)

【暴击示例】
敏捷30 → 暴击率15%，暴击伤害150%+6% = 156%
敏捷50 → 暴击率25%，暴击伤害150%+10% = 160%

【暴击规则】
├─ 暴击时，伤害数字显示为黄色
└─ 暴击可触发"暴击回能"词缀
```

### 3.4 伤害公式

```
【物理伤害公式】
原始伤害 = 物理攻击 × 技能倍率 × 时机加成

物理攻击 = 体质 × 2 + 武器基础攻击 + 装备加成
最终伤害 = (原始伤害 × 伤害减免) - 防御力

【奥术伤害公式】
原始伤害 = 魔法攻击 × 技能倍率 × 时机加成

魔法攻击 = 精神 × 2 + 武器基础攻击 + 装备加成
最终伤害 = (原始伤害 × 伤害减免) - 魔抗

【伤害减免】
├─ 来源：装备、技能、状态
├─ 上限：90%（防止伤害为0或负数）
└─ 基础值：0%
示例：
  物理伤害减免30% → 受到物理伤害×70%
```

### 3.5 时机加成

```
【行动时机判定】
├─ 完美时机(100%)：ATB槽刚好满时行动 → 伤害×1.15
├─ 良好时机(90-99%)：接近满槽 → 伤害×1.05
├─ 普通时机(70-89%)：正常行动 → 伤害×1.0
└─ 仓促时机(<70%)：槽未满就行动 → 伤害×0.8

【视觉反馈】
├─ 完美时机：金色光芒特效，伤害数字+15%显示
├─ 良好时机：蓝色光芒，伤害数字+5%显示
├─ 普通时机：无特效
└─ 仓促时机：红色闪烁，伤害数字-20%显示
```

## 4. 防御与护盾

### 4.1 防御属性

```
【防御力来源】
├─ 基础防御：体质 × 1.5
├─ 装备防御：防具提供
└─ 状态加成：技能/词缀

【防御效果】
├─ 物理减伤 = 防御 / (防御 + 100)
├─ 100防御 → 50%物理减伤
├─ 200防御 → 67%物理减伤
└─ 300防御 → 75%物理减伤（边际递减）

【魔抗效果】
├─ 魔抗来源：精神 × 1.5 + 装备
├─ 奥术减伤 = 魔抗 / (魔抗 + 100)
└─ 同物理减伤公式
```

### 4.2 护盾系统

```
【护盾获取】
├─ 铁壁：护盾值 = 防御 × 3
├─ 法术护盾：护盾值 = 魔法攻击 × 2
└─ 装备词缀：各种护盾效果

【护盾机制】
├─ 护盾优先于HP承受伤害
├─ 护盾存在时，可触发"铁壁·荆棘"词缀
├─ 护盾破碎时，不触发受击效果
└─ 护盾持续时间：通常3-4秒

【护盾优先级】
护盾 → 伤害减免% → 防御减伤 → 最终HP
```

## 5. 战斗流程

### 5.1 战斗开始

```
【进入战斗】
1. 玩家遭遇敌人（踩到敌人触发点/主动攻击）
2. 屏幕渐变（0.5秒）切换到战斗场景
3. 初始化：双方ATB槽从0%开始蓄力
4. 显示战斗UI（敌人血条、技能栏、状态栏）
5. 玩家开始操作
```

### 5.2 回合循环

```
【回合循环】
while (战斗未结束) {
    // 1. ATB蓄力阶段
    所有角色和敌人ATB槽持续蓄力

    // 2. 行动选择阶段
    if (玩家ATB满) {
        显示技能菜单
        玩家选择行动
        执行行动
    }

    if (敌人ATB满) {
        AI选择行动
        执行行动
    }

    // 3. 结算阶段
    判定战斗结果（HP≤0？逃跑？）
}
```

### 5.3 战斗结束

```
【胜利条件】
├─ 所有敌人HP归0
└─ 触发奖励结算

【失败条件】
├─ 玩家HP归0
└─ 触发死亡结算

【逃跑】
├─ 成功率：50% + 敏捷差值%
├─ 失败：敌人获得一次额外攻击
├─ 成功后返回据点，损失20%当前星尘
└─ 掉落：可能损失部分掉落物品
```

## 6. BOSS战斗

### 6.1 BOSS机制

```
【BOSS特性】
├─ 10倍普通怪HP
├─ 2倍伤害倍率
├─ 阶段机制：BOSS有2-3个阶段
│   ├─ 阶段转换：HP降到50%/25%时触发
│   └─ 阶段效果：攻击模式改变/召唤小怪
├─ 低命中率：BOSS攻击最低20%命中率
└─ 不可逃跑：BOSS战强制战斗

【BOSS阶段示例】
第一阶段（100%-50%HP）：普通攻击+技能A
第二阶段（50%-25%HP）：技能B+召唤小怪
第三阶段（25%-0%HP）：技能C连续释放
```

### 6.2 BOSS掉落

```
【BOSS掉落】
├─ 星尘：100%掉落，50个
├─ 装备：100%掉落，1-2件
├─ 材料：100%掉落，1-3个
├─ 记忆碎片：20%概率
├─ 永久增幅物品：10%概率
└─ 套装组件：30%概率
```
