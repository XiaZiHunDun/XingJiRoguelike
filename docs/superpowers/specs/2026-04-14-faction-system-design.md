## 1. 四大阵营

### 1.1 阵营总览

| 阵营 | 理念 | 颜色 | 根据地 | 特色 |
|------|------|------|--------|------|
| **星火殿** | 追求力量，超越凡人 | 红色/金色 | 沙海回声-火焰祭坛 | 高阶技能书、稀有词缀 |
| **寒霜阁** | 掌控秘法，秩序与平衡 | 蓝色/白色 | 霜棘王庭-冰晶圣所 | 秘法技能书、控制技能 |
| **机魂教** | 机械至上，科技复兴 | 橙色/铜色 | 机魂废土-机械圣殿 | 机械强化、ATB强化 |
| **守墓人** | 守护秘密，阻止外人离去 | 黑色/银色 | 太初核心-虚空祭坛 | 唯一性装备、虚空道具 |

### 1.2 阵营定位

```
【主要阵营】（Phase 1 包含）
├─ 星火殿：物理/攻击强化
├─ 寒霜阁：魔法/控制强化
├─ 机魂教：ATB/机械强化
└─ 守墓人：虚空/唯一性装备

【守墓人特殊说明】
├─ 唯一敌对阵营
├─ 击杀守墓人成员可获得贡献
└─ 其他阵营都可以攻击守墓人获取奖励
```

## 2. 阵营加入与退出

### 2.1 加入机制

```
【加入条件】
├─ 玩家可选择加入任意一个阵营
├─ 加入方式：在阵营据点与NPC对话
├─ 无需任务，直接加入
└─ 同时只能拥有一个阵营身份

【加入限制】
├─ 玩家境界达到要求后即可加入
├─ 各阵营入门要求不同（可选设计）
└─ 阵营选择是永久的，无法退出

【重要提示】
└─ 加入后无法退出，请慎重选择阵营
```

### 2.2 退出与更换

```
【退出阵营】
├─ 无法退出（永久绑定）
├─ 玩家一旦加入某个阵营，将永远属于该阵营
└─ 只能通过更换角色来更换阵营

【更换阵营】
├─ 无法直接更换
├─ 必须通过退出游戏创建新角色
└─ 新角色可以重新选择阵营

【特殊说明】
├─ 这是Roguelike的永久选择机制
├─ 玩家需要慎重选择阵营
└─ 阵营选择影响Build方向和游戏体验
```

## 3. 阵营关系系统

### 3.1 关系等级

```
【关系等级】（初始值为50/100）
├─ 友好（80-100）：正常状态
├─ 中立（50-79）：略有下降
├─ 冷淡（20-49）：明显敌意
└─ 敌对（0-19）：高度敌意

【关系影响】
├─ 80-100：商店价格正常，任务奖励正常
├─ 50-79：商店价格+20%，任务奖励不变
├─ 20-49：商店价格+50%，任务奖励-10%
└─ 0-19：商店价格+100%，任务奖励-20%，无法接取高级任务
```

### 3.2 关系变化

```
【降低关系的行为】
├─ 攻击该阵营成员：-5~20（根据敌人等级）
├─ 击杀该阵营成员：-10~30（根据敌人等级）
└─ 拒绝该阵营任务：-5

【恢复关系的方式】
├─ 完成该阵营任务：+5~15
├─ 捐献物资：少量恢复
├─ 时间流逝：每周自动+5
└─ 特殊道具（赎罪符）：+20~50
```

### 3.3 阵营间关系

```
【阵营间关系】
├─ 所有阵营之间是竞争关系，非敌对
├─ 玩家攻击某阵营，该阵营对玩家关系下降
├─ 不攻击的阵营保持中立或友好
└─ 守墓人是唯一对所有玩家都敌对的阵营

【守墓人特殊】
├─ 守墓人对所有玩家初始即为敌对
├─ 击杀守墓人成员，所有玩家都可获得奖励
└─ 无法加入守墓人阵营
```

## 4. 贡献值系统

### 4.1 贡献值获取

```
【获取途径】
├─ 完成势力任务：+10~100贡献（根据难度）
├─ 击杀守墓人成员：+5~50贡献
├─ 捐献物资：+1~10贡献/物品
└─ 每日首次登录：+5贡献

【贡献值上限】
└─ 无上限，但等级有上限
```

### 4.2 贡献等级

| 等级 | 贡献值范围 | 解锁内容 |
|------|------------|----------|
| 学徒 | 0-100 | 基础装备、基础物资 |
| 正式成员 | 101-500 | 中级装备、中级物资 |
| 精英成员 | 501-1500 | 高级装备、高级物资、稀有词缀 |
| 长老 | 1501-5000 | 终极装备、终极技能、专属词缀 |
| 掌门 | 5000+ | 唯一性装备制作图、虚空道具 |

### 4.3 贡献值消耗

```
【消耗途径】
├─ 兑换势力装备：100~500贡献
├─ 兑换势力物资：50~200贡献
├─ 兑换势力技能：200~1000贡献
└─ 兑换唯一性装备制作图：500~2000贡献
```

## 5. 每日任务系统

### 5.1 任务池机制

```
【任务池规则】
├─ 每个阵营有独立的20个任务池
├─ 每日凌晨刷新20个任务
├─ 玩家只能看到自己阵营的任务
├─ 其他阵营任务不可见
└─ 任务池独立，互不影响

【任务刷新】
├─ 每日凌晨自动刷新
├─ 手动刷新：消耗星尘（10星尘/次）
└─ 刷新后重新抽取20个任务
```

### 5.2 任务类型

```
【任务分类】
├─ 击杀任务：击败X个敌人
├─ 收集任务：收集X个材料
├─ 探索任务：到达指定位置
└─ 挑战任务：击败精英/BOSS

【任务难度】
├─ 简单：击败5个低级敌人，奖励10-20贡献
├─ 普通：击败10个敌人，奖励30-50贡献
├─ 困难：击败精英/BOSS，奖励80-150贡献
└─ 噩梦：挑战高等级区域BOSS，奖励200-500贡献
```

### 5.3 任务奖励

```
【奖励构成】
├─ 贡献值：主要奖励
├─ 星尘：少量
└─ 材料：概率掉落

【任务失败】
├─ 放弃任务：消耗任务次数
├─ 任务完成前退出：不消耗任务次数
└─ 可重新抽取新任务（消耗刷新次数）
```

## 6. 阵营商店系统

### 6.1 商店功能

```
【商店商品】
├─ 装备：阵营专属装备
├─ 物资：消耗品、材料
├─ 技能：阵营专属技能书
├─ 特殊：唯一性装备制作图（需长老等级）
└─ 增幅器：绿色品质（初）

【商店刷新】
├─ 每次进入据点刷新
├─ 手动刷新消耗星尘
└─ 刷新价格：10星尘/次
```

### 6.2 商店价格

```
【基础价格公式】
实际价格 = 基础价格 × 关系系数

【关系系数】
├─ 友好（80-100）：×1.0
├─ 中立（50-79）：×1.2
├─ 冷淡（20-49）：×1.5
└─ 敌对（0-19）：×2.0

【示例】
一件基础价格100贡献的装备：
├─ 友好状态：100贡献
├─ 中立状态：120贡献
├─ 冷淡状态：150贡献
└─ 敌对状态：200贡献
```

### 6.3 商店解锁

```
【按贡献等级解锁】
├─ 学徒：基础装备、基础物资
├─ 正式成员：中级装备、中级物资
├─ 精英成员：高级装备、高级物资、稀有词缀
├─ 长老：终极装备、终极技能、专属词缀、绿色增幅器
└─ 掌门：蓝色/紫色增幅器、唯一性装备制作图
```

## 8. 势力敌人交互

### 8.1 遭遇势力敌人

```
【刷新规则】
├─ 每张地图15%概率刷新势力敌人
├─ 阵营随机：星火殿/寒霜阁/机魂教/守墓人
├─ 刷新数量：1-2个
├─ 位置：地图内随机
└─ 等级：跟随地图等级
```

### 8.2 交互选择

```
【玩家选择】
├─ 攻击：进入战斗，击杀后获得奖励
├─ 忽略：不攻击，敌人也不会攻击玩家
└─ 敌人不会主动攻击（除非玩家先攻击）

【击杀奖励】
守墓人：
├─ 贡献值：+5~50
├─ 星尘：20-40个
├─ 装备：1件（57-70级区域）
└─ 守墓人徽记

其他阵营：
├─ 贡献值：无
├─ 星尘：10-20个
├─ 装备：1件
├─ 该阵营关系：-10~-30
└─ 可能掉落该阵营物资（低概率）
```

### 8.3 关系惩罚

```
【关系下降】
├─ 攻击该阵营成员：-5
├─ 击杀该阵营成员：-20
└─ 连续击杀：关系下降累积

【惩罚影响】
├─ 关系50-79：商店价格+20%
├─ 关系20-49：商店价格+50%，任务奖励-10%
└─ 关系0-19：商店价格+100%，任务奖励-20%，无法接取高级任务
```
