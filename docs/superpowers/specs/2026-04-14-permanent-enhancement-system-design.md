> 版本：1.0

> 状态：待评审

## 1. 系统概述

```
【系统定位】
├─ 局外永久生效的Meta进度
├─ 死亡不消失，多周目可继承
└─ 增强角色基础属性

【获取途径】
├─ 怪物掉落：普通怪/精英/BOSS
├─ 地图采集：各区域的特殊采集点
├─ 记忆碎片兑换
└─ 特殊事件/成就

【物品分类】
├─ 淬体液：体质属性加成
├─ 聚魂露：精神属性加成
└─ 疾风露：敏捷属性加成

【品质等级】
├─ 初：+1属性（最容易获得）
├─ 中：+2属性（中等难度）
└─ 极：+5属性（最难获得）
```

## 3. 使用规则

### 3.1 独立次数限制

```
【每角色独立计算】
├─ 星际战士：淬体液·初最多使用10次
├─ 奥术师：淬体液·初最多使用10次
└─ 两个角色互不影响

【示例】
星际战士已使用淬体液·初 8次
├─ 剩余：2次使用机会
├─ 奥术师：仍有10次使用机会
└─ 新游戏创建新角色：重新计算10次
```

### 3.2 使用效果

```
【属性加成计算】
淬体液·初 × 10 = 体质+10
淬体液·中 × 10 = 体质+20
淬体液·极 × 10 = 体质+50

【最大加成】（每角色）
├─ 体质：+80（10初+10中+10极）
├─ 精神：+80（10初+10中+10极）
└─ 敏捷：+80（10初+10中+10极）
```

### 3.3 使用时机

```
【使用时机】
├─ 据点使用：在据点的角色面板使用
├─ 背包使用：从背包直接使用
└─ 局内使用：战斗中也可以使用

【使用限制】
├─ 必须满足使用条件（如果有）
└─ 使用后立即生效，无需确认
```

## 4. 获取方式

### 4.1 初级获取（普通怪/精英/采集）

### 4.2 中级获取（BOSS）

```
【怪物掉落概率】
├─ 普通怪：1%概率掉落
├─ 精英怪：5%概率掉落
├─ 掉落类型：淬体液·初/聚魂露·初/疾风露·初
└─ 掉落等级：跟随地图等级

【地图采集】
├─ 采集点：各区域的特殊采集点
├─ 采集物类型：矿石、植物、材料、药品、永久增幅物品等
├─ 采集概率：10%概率获得初级永久增幅物品
├─ 采集类型：随机获得淬体液/聚魂露/疾风露·初
└─ 采集次数：每个采集点可采集1次

【采集系统详情】
└─ 详见：物资与采集系统文档

【掉落规则】
├─ 所有区域都可能掉落初级物品
├─ 初级物品主要在1-28级区域掉落
└─ 掉落为绑定状态
```

### 4.2 中级获取（BOSS/采集）

```
【BOSS掉落概率】
├─ 区域BOSS：20%概率掉落
├─ 掉落类型：淬体液·中/聚魂露·中/疾风露·中
└─ 掉落等级：跟随BOSS等级

【地图采集】
├─ 采集点：各区域的特殊采集点（29-42级以上区域）
├─ 采集物类型：矿石、植物、材料、药品、永久增幅物品等
├─ 采集概率：5%概率获得中级永久增幅物品
├─ 采集类型：随机获得淬体液/聚魂露/疾风露·中
└─ 采集次数：每个采集点可采集1次

【掉落规则】
├─ 所有区域BOSS都可能掉落中级物品
├─ 太初之子（最终BOSS）：掉落概率更高（30%）
└─ 掉落为绑定状态
```

### 4.3 极级获取（记忆碎片/特殊事件）

```
【记忆碎片兑换】
├─ 兑换NPC：各据点的特殊NPC
├─ 兑换价格：10记忆碎片/个
├─ 兑换数量：每周限量（可选）
└─ 兑换类型：淬体液·极/聚魂露·极/疾风露·极

【特殊事件】
├─ 隐藏宝箱：各区域可能有隐藏宝箱
├─ 成就奖励：达成特定成就获得
├─ 任务奖励：某些支线任务奖励
└─ 节日活动：（如有）特殊活动获取
```

## 5. 物品管理

### 5.1 物品堆叠

```
【堆叠规则】
├─ 同类型物品可以堆叠
├─ 堆叠上限：99个
└─ 不同品质分开堆叠

【示例】
背包中：
├─ 淬体液·初 × 15
├─ 淬体液·中 × 8
├─ 聚魂露·极 × 3
└─ 疾风露·中 × 12
```

### 5.2 物品交易

```
【交易规则】
├─ 永久增幅物品为绑定状态
├─ 不可出售给商店
├─ 不可丢弃
└─ 不可交易给其他角色
```

## 7. 多周目继承

### 7.1 继承规则

```
【Roguelike继承机制】
├─ 新游戏开始时，可以选择继承永久增幅
├─ 继承后，永久增幅从新角色开始生效
└─ 物品不会继承（已使用的次数不可恢复）

【不继承情况】
├─ 玩家选择不继承
├─ 创建全新角色（不勾选继承）
└─ 部分平台/模式可能不支持继承
```

### 7.2 继承界面


## 8. 数值影响

### 8.1 属性加成示例

```
【星际战士（50级）无增幅】
├─ 体质：130
├─ 精神：75
└─ 敏捷：80

【星际战士（50级）满增幅】
├─ 体质：130 + 80 = 210 (+61%)
├─ 精神：75 + 80 = 155 (+106%)
└─ 敏捷：80 + 80 = 160 (+100%)
```

### 8.2 战斗数值影响

```
【ATB速度计算】
基础ATB速度 = 100 + 敏捷 × 3

无增幅（敏捷80）：
└─ ATB速度 = 100 + 80 × 3 = 340

满增幅（敏捷160）：
└─ ATB速度 = 100 + 160 × 3 = 580
（速度溢出：580 - 200 = 380，伤害+380%）

【HP计算】
HP = 100 + 体质 × 8

无增幅（体质130）：
└─ HP = 100 + 130 × 8 = 1140

满增幅（体质210）：
└─ HP = 100 + 210 × 8 = 1780 (+56%)
```

## 9. 平衡性考虑

### 9.1 增幅上限

```
【属性上限】
├─ 体质：+80（最大）
├─ 精神：+80（最大）
├─ 敏捷：+80（最大）
└─ 总计：+240属性点

【设计目的】
├─ 防止属性膨胀过快
├─ 保持Roguelike的核心体验
└─ 多周目仍有成长空间但不是无限
```

### 9.2 获取难度曲线

```
【获取难度】
├─ 初级（+1）：容易大量获取
├─ 中级（+2）：需要击杀BOSS积累
└─ 极（+5）：稀有，需要记忆碎片兑换

【时间投入】
├─ 初级满增幅：约10-20局游戏
├─ 中级满增幅：约20-40局游戏
└─ 极满增幅：约50+记忆碎片兑换
```
