## 1. 境界概述

### 1.1 境界等级结构

```
【境界划分】（Phase 1 仅开放至星火境）

凡人境（1-10级）
├─ 描述：初入修炼，吸收星尘
├─ 特征：基础属性成长
└─ 特权：初始开启1个增幅器格子

感应境（11-20级）
├─ 描述：感应星尘粒子
├─ 特征：解锁元素亲和
└─ 特权：星尘感应（地图显示隐藏采集点）

聚尘境（21-30级）
├─ 描述：星尘凝聚成型
├─ 特征：技能强度提升
└─ 特权：开启第2个增幅器格子

凝核境（31-40级）
├─ 描述：星尘核蜕变为星尘之躯
├─ 特征：所有属性大幅提升
└─ 特权：开启第3个增幅器格子

星火境（41-50级）
├─ 描述：星尘燃烧，化为星火
├─ 特征：所有属性达到顶峰
└─ 特权：终极形态（终极技能变体）

【后续境界】（Phase 2+ 启用）
└─ 脉冲境 → 裂变境 → 黑洞境 → 创世境
```

### 1.2 境界特权总结

| 境界 | 等级范围 | 增幅器格子 | 境界特权 |
|------|----------|------------|----------|
| 凡人身 | 1-10级 | 1个 | 无 |
| 感应境 | 11-20级 | 1个 | 星尘感应 |
| 聚尘境 | 21-30级 | 2个 | 无（仅解锁格子） |
| 凝核境 | 31-40级 | 3个 | 无（仅解锁格子） |
| 星火境 | 41-50级 | 3个 | 终极形态 |

## 2. 突破条件

### 2.1 突破方式

```
【属性突破】
├─ 每个境界有属性要求
├─ 满足属性要求即可突破
├─ 无需战斗/任务
└─ 在据点的境界面板突破

【突破流程】
1. 达到当前境界等级上限
2. 检查属性是否达标
3. 属性达标 → 可突破
4. 消耗星尘完成突破
5. 进入下一境界
```

### 2.2 各境界突破属性要求

| 突破方向 | 突破要求 | 解锁增幅器格子 | 获得境界特权 |
|----------|----------|----------------|--------------|
| 凡人身→感应境 | 体质≥25，精神≥20，敏捷≥20 | 无 | 星尘感应 |
| 感应境→聚尘境 | 体质≥45，精神≥35，敏捷≥35 | 第2个格子 | 无 |
| 聚尘境→凝核境 | 体质≥70，精神≥55，敏捷≥55 | 第3个格子 | 无 |
| 凝核境→星火境 | 体质≥100，精神≥80，敏捷≥80 | 无 | 终极形态 |

### 2.3 突破消耗

```
【突破消耗】
├─ 凡人身 → 感应境：50星尘
├─ 感应境 → 聚尘境：100星尘
├─ 聚尘境 → 凝核境：150星尘
├─ 凝核境 → 星火境：200星尘
└─ 突破后，当前境界重置为1级
```

### 2.4 突破示例

```
【玩家当前状态】
├─ 境界：凡人境10级
├─ 体质：28，精神：22，敏捷：25
└─ 星尘：150

【突破判定】
├─ 突破要求：体质≥25，精神≥20，敏捷≥20
├─ 当前属性：体质28✓，精神22✓，敏捷25✓
└─ 满足条件，可突破

【突破消耗】
├─ 消耗星尘：50
├─ 突破后：星尘剩余100
├─ 新境界：感应境1级
└─ 新特权：第2个增幅器格子开启
```

## 3. 境界属性成长

### 3.1 每级属性成长

```
【每级属性成长】
├─ 体质：+3/级
├─ 精神：+2.5/级
├─ 敏捷：+2.5/级
├─ HP：+8/级（体质×8）
└─ 能量：+0.5/级（精神×0.5）

【ATB速度】
└─ 基础ATB速度 = 100 + 敏捷×3 + 装备加成
```

| 凡人身 | 10级 | 67 | 52.5 | 52.5 | 736 | 30.25 |
| 感应境 | 20级 | 97 | 75 | 75 | 976 | 42.5 |
| 聚尘境 | 30级 | 127 | 97.5 | 97.5 | 1216 | 53.75 |
| 凝核境 | 40级 | 157 | 120 | 120 | 1456 | 65 |

## 4. 增幅器系统

### 4.1 增幅器格子

```
【增幅器格子】
├─ 总共3个格子
├─ 格子1：初始开启（凡人身）
├─ 格子2：聚尘境（21级）解锁
├─ 格子3：凝核境（31级）解锁
└─ 格子4+：后续境界可能开启

【格子解锁】
├─ 凡人身（1级）：1个格子
├─ 感应境（11级）：1个格子（无新格子）
├─ 聚尘境（21级）：2个格子
├─ 凝核境（31级）：3个格子
└─ 星火境（41级+）：3个格子
```

### 4.3 增幅器获取

```
【获取方式】
├─ 贡献兑换：阵营商店兑换
├─ 任务奖励：主线/支线任务
├─ BOSS掉落：概率掉落
└─ 特殊事件：隐藏宝箱等

【贡献价格】
├─ 绿色（初）：300贡献
├─ 蓝色（中）：600贡献
└─ 紫色（极）：1200贡献
```

### 4.4 增幅器特性

```
【共同特性】
├─ 科技装备，不带技能和词缀
├─ 品质：绿色→蓝色→紫色（由低到高）
├─ 不可出售
├─ 不可重铸
├─ 可随时更换
└─ 死亡后保留
```

### 4.5 增幅器叠加示例

```
【示例：星火境50级角色】
基础属性：体质187，精神142.5，敏捷142.5

装备增幅器：
├─ 格子1：体质增幅器·极（×2.5）
├─ 格子2：HP增幅器·极（×2.0）
└─ 格子3：防御增幅器·极（×2.0）

计算后：
├─ 体质：187 × 2.5 = 467.5
├─ HP：(100 + 187×8) × 2.0 = 1596
└─ 防御：281 × 2.0 = 562
```

## 5. 境界与穿戴要求

### 5.1 境界影响装备穿戴

```
【穿戴要求】
├─ 装备有境界等级要求
├─ 玩家境界必须达到或超过装备要求
└─ 不满足要求的装备无法穿戴

【装备境界等级】
├─ 凡人身装备（1-10级）：凡人可穿
├─ 感应境装备（11-20级）：需要感应境+
├─ 聚尘境装备（21-30级）：需要聚尘境+
├─ 凝核境装备（31-40级）：需要凝核境+
└─ 星火境装备（41-50级）：需要星火境
```
