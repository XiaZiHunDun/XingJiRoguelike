## 1. 敌人分类

### 1.1 三种敌人类型

```
【普通敌人：原生异兽】
├─ 来源：文明灭亡后占据遗迹的本土生物
├─ 数量：每区域3-5种基础类型
├─ 特点：基础AI，攻击模式单一
└─ 掉落：星尘、装备（概率）

【精英敌人：古代守卫】
├─ 来源：文明灭亡后仍在运作的守护构造体
├─ 数量：每区域1-2种独特类型
├─ 特点：更强AI，多阶段攻击
├─ 掉落：星尘（高概率）、装备（高概率）、材料
└─ 刷新：随机位置刷新

【BOSS：区域之主】
├─ 来源：每个区域的终极威胁
├─ 数量：每区域1个
├─ 特点：独特阶段机制、召唤小怪
├─ 掉落：套装组件（概率）、记忆碎片（概率）、永久增幅物品（概率）
└─ 位置：固定在地图5
```

## 4. 精英敌人

### 4.1 精英刷新机制

```
【随机刷新规则】
├─ 每张地图有30%概率刷新1个精英
├─ 20%概率刷新2个精英
├─ 精英位置：地图内随机
├─ 精英等级：跟随地图等级
└─ 精英类型：每区域1-2种

【刷新时机】
├─ 玩家进入地图时判定
├─ 精英不会在玩家已清空的区域刷新
└─ 已刷新的精英死亡后不会刷新新的
```

### 4.2 势力敌人（各阵营成员）

```
【势力敌人刷新机制】
├─ 每张地图有15%概率刷新各阵营成员
├─ 刷新阵营：星火殿/寒霜阁/机魂教/守墓人（随机）
├─ 守墓人是敌对阵营（击杀必定获得奖励）
├─ 其他阵营：友好/中立状态（攻击可能影响阵营关系）
├─ 势力敌人位置：地图内随机
├─ 势力敌人等级：跟随地图等级
├─ 刷新数量：1-2个
└─ 刷新时机：玩家进入地图时判定

【各阵营敌人类型】
星火殿：
├─ 星火战士：物理攻击型
├─ 星火法师：火系魔法攻击
└─ 星火精英：更强力

寒霜阁：
├─ 寒霜修士：冰系魔法攻击
├─ 寒霜刺客：高速攻击型
└─ 寒霜精英：更强力

机魂教：
├─ 机魂战士：物理攻击型
├─ 机魂工程师：召唤机械助手
└─ 机魂精英：更强力

守墓人：
├─ 守墓人战士：物理攻击型
├─ 守墓人法师：虚空魔法攻击
└─ 守墓人精英：更强力

【玩家选择】
├─ 可攻击任意势力敌人
│   ├─ 守墓人：击杀获得贡献值（+5~50）
│   ├─ 其他阵营：攻击会降低该阵营关系
│   └─ 不攻击则无事发生
└─ 势力敌人不会主动攻击玩家（除非玩家先攻击）

【势力敌人掉落】
守墓人：
├─ 贡献值：+5~50
├─ 星尘：20-40个
├─ 装备：1件（57-70级区域）
└─ 特殊道具：守墓人徽记

其他阵营：
├─ 星尘：10-20个
├─ 装备：1件
└─ 可能掉落该阵营物资（低概率）
```

### 4.4 精英战斗机制

```
【沙神守卫】
├─ HP：普通怪的3倍
├─ 技能1：召唤2只小沙蝎（每10秒一次）
├─ 技能2：沙暴，对扇形区域造成伤害
└─ 弱点：火系伤害×1.5

【冰魔像】
├─ HP：普通怪的3倍
├─ 被动：冰冻光环，周围敌人减速20%
├─ 技能：冰刺穿刺，附带冻结效果
└─ 弱点：火系伤害×1.5

【机魂巨人】
├─ HP：普通怪的3倍
├─ 技能1：召唤2个小机器人
├─ 技能2：激光扫描，需要躲避
└─ 弱点：冰系伤害×1.5

【太古魔像】
├─ HP：普通怪的5倍
├─ 阶段1：普通攻击
├─ 阶段2（HP<50%）：激活护盾，吸收伤害
├─ 阶段3（HP<25%）：全屏能量攻击
└─ 弱点：物理伤害×1.5
```

### 5.2 BOSS通用机制

```
【BOSS特性】
├─ HP：普通怪的10倍
├─ 伤害：普通怪的2倍
├─ 防御：普通怪的2倍
├─ 最低命中率：20%（玩家攻击BOSS至少20%命中）
├─ 阶段机制：HP降到50%/25%时转换阶段
└─ 不可逃跑

【BOSS阶段示例】

阶段1（100%-50%HP）：
├─ 普通攻击+技能A
└─ 约10秒一个循环

阶段2（50%-25%HP）：
├─ 技能B+召唤小怪
└─ 攻击频率提高

阶段3（25%-0%HP）：
├─ 技能C连续释放
└─ 进入狂暴状态
```

## 7. 敌人AI

### 7.1 普通敌人AI

```
【AI行为模式】
├─ 接近玩家
├─ 使用普通攻击
├─ 血量低时继续战斗（不逃跑）
└─ 攻击间隔：约2-3秒

【仇恨机制】
├─ 优先攻击最近的角色
├─ 受到攻击后仇恨转移
└─ 仇恨持续时间：5秒
```

### 7.2 精英敌人AI

```
【AI行为模式】
├─ 接近玩家
├─ 使用技能（循环）
├─ 召唤小怪
├─ 血量低时使用特殊技能
└─ 攻击间隔：约1.5-2秒

【技能循环示例：沙神守卫】
1. 普通攻击
2. 普通攻击
3. 召唤小沙蝎
4. 普通攻击
5. 沙暴
6. 循环
```

### 7.3 BOSS AI

```
【AI行为模式】
├─ 追踪玩家
├─ 使用阶段技能
├─ 召唤小怪
├─ 血量触发阶段转换
└─ 攻击间隔：约1-1.5秒

【阶段转换示例：机魂霸主】
阶段1（100%-50%HP）：
├─ 激光扫描（每15秒）
├─ 普通攻击
└─ 召唤机器人（每20秒）

阶段2（50%-25%HP）：
├─ 激光扫描（每10秒）
├─ 能量护盾
└─ 召唤机器人（每15秒）

阶段3（25%-0%HP）：
├─ 激光扫描（每5秒）
├─ 能量护盾（更频繁）
├─ 召唤机器人（每10秒）
└─ HP<10%：自爆
```
