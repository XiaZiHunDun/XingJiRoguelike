## 1. 设计原则

| 原则 | 实现要求 |
|------|----------|
| **直觉导航** | 重要功能≤2次点击可达 |
| **节奏流畅** | 高频面板动画≤0.1s |
| **层级清晰** | 状态机无歧义，面板打开时锁定切换 |
| **反馈明确** | 每次操作有视觉/音效反馈 |
| **防御性设计** | 数据操作必须有校验和容错 |

## 7. 数据流设计

### 7.1 架构

```
RunState (autoload - 数据中心)
    │
    ├── current_character_id
    ├── current_zone
    ├── current_level
    └── current_map_nodes: Array[MapNode]
            │
            ▼
    GameScene (主控制器)
            │
            ├── HubScene ──────────────────┐
            │                              ↓
            ├── MapScene ─────────────────→ RunState.current_map_nodes
            │                              ↓
            ├── BattleScene ─────────────→ RunState.current_map_nodes[i]
            │                              ↓
            └── PanelScenes ←────────────── RunState (读取/更新)
```

### 7.2 MapScene初始化流程

```
GameScene._show_map()
    │
    ├── if current_map_nodes.is_empty():
    │       RunState.generate_zone_map()
    │
    ▼
MapScene 实例化
    │
    ▼
MapScene._ready()
    │
    ├── 动态实例化节点到 MapContainer
    ├── 连接 node_clicked 信号
    ├── await process_frame (等待子节点ready)
    ├── initialized.emit()
    │
    ▼
MapScene.setup_map(zone) [外部调用]
    │
    ├── 验证 map_nodes
    ├── 清空旧节点
    └── 遍历 map_nodes 动态绑定数据
```

### 7.3 节点数据绑定规范

**MapScene.gd:**
```gdscript
const MAP_NODE_SCENE = preload("res://scenes/map/node_scene.tscn")

func setup_map(zone: ZoneDefinition) -> void:
    var map_nodes = RunState.current_map_nodes
    if map_nodes.is_empty():
        RunState.generate_zone_map(zone)
        map_nodes = RunState.current_map_nodes

    # 清理并重建
    for child in $MapContainer.get_children():
        child.queue_free()

    for i in map_nodes.size():
        var node_ui = MAP_NODE_SCENE.instantiate()
        node_ui.set_node_data(map_nodes[i])
        node_ui.node_clicked.connect(_on_node_clicked)
        $MapContainer.add_child(node_ui)
```

**NodeScene.gd:**
```gdscript
func set_node_data(data: MapNode) -> void:
    assert(data != null, "节点数据不能为空")
    node_data = data
    refresh_ui()

func refresh_ui():
    if node_data == null:
        mouse_filter = MOUSE_FILTER_IGNORE
        return
    # 更新显示...
```
