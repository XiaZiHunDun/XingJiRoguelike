# 星际Roguelike Playtest验证清单

> 测试日期：待测试

---

## 游戏流程

```
主菜单 → 角色选择 → 据点 → 地图 → 战斗 → 胜利/失败 → 据点
```

---

## P0 - 核心体验（必须通过）

### 1. 主菜单
- [ ] 游戏启动无崩溃
- [ ] "开始游戏" 按钮可点击
- [ ] "继续游戏" 按钮（如有存档）

### 2. 角色选择
- [ ] 星际战士选项显示正确
- [ ] 奥术师选项显示正确
- [ ] 点击确认后进入据点

### 3. 据点 (Hub)
- [ ] 据点界面正常显示
- [ ] 各功能按钮可点击：
  - [ ] 装备管理
  - [ ] 背包
  - [ ] 任务
  - [ ] 角色状态
  - [ ] 地图
  - [ ] 商店
  - [ ] 永久增幅
  - [ ] 退出

### 4. 地图探索
- [ ] 地图正常加载
- [ ] 节点显示正确（普通/精英/BOSS/采集/商店）
- [ ] 点击节点进入对应事件

### 5. 战斗系统
- [ ] ATB槽正常填充
- [ ] 技能按钮可用
- [ ] 普通攻击正常
- [ ] 技能释放消耗能量
- [ ] 完美时机判定（≥90%显示）
- [ ] 子弹时间触发
- [ ] 时砂可用
- [ ] 敌人AI正常（攻击、特殊技能）
- [ ] 战斗结束判定正确

### 6. 胜负判定
- [ ] HP归零 → 死亡界面 → 回到主菜单
- [ ] 敌人全灭 → 胜利界面 → 掉落奖励

---

## P1 - 主要系统（应该通过）

### 7. 装备系统
- [ ] 装备显示正确（名称、属性、词缀）
- [ ] 穿戴要求检查（属性/境界/技能等级）
- [ ] 颜色提示正确（满足=绿，不满足=红）
- [ ] 套装效果触发

### 8. 词缀系统
- [ ] 词缀显示正确
- [ ] 恒定型词缀生效
- [ ] 触发型词缀在条件满足时触发
- [ ] 形态改变词缀修改技能形态
- [ ] 共鸣效果正确计算

### 9. 境界系统
- [ ] 境界显示正确
- [ ] 境界突破条件检查
- [ ] 突破消耗星尘正确
- [ ] 突破后属性提升

### 10. 材料/采集系统
- [ ] 采集点显示
- [ ] 采集次数限制（3次后冷却）
- [ ] 材料获得

### 11. 存档系统
- [ ] 自动存档触发（境界突破、区域解锁）
- [ ] 快速存档（S键）
- [ ] 读档正确恢复状态

### 12. 永久增幅
- [ ] 增幅器显示正确
- [ ] 使用后属性提升
- [ ] 10次上限检查

- [ ] 任务奖励发放

- [ ] 唯一装备lore显示
