# 技能设计文档
> Phase 0/1 技能数据设计

---

## 一、技能设计原则

1. **技能来源**：装备决定技能池
2. **能量消耗**：1-3费为主，终极技3-4费
3. **元素属性**：技能必须带有元素类型
4. **连携设计**：同源技能可触发连携

---

## 二、技能分类

| 类型 | 消耗 | 定位 |
|------|------|------|
| 攻击技能 | 1-3费 | 主要输出 |
| 防御技能 | 0-2费 | 格挡/护盾 |
| 辅助技能 | 0-2费 | Buff/Debuff |
| 终极技 | 3-4费 | 改变战局 |

## 七、技能详细效果设计

### 7.1 伤害计算公式

```
最终伤害 = 基础伤害 × ATB时机加成 × (1 + 动能加成) × 元素克制 × (1 + 暴击加成)
```

| 加成类型 | 触发条件 | 数值 |
|---------|---------|------|
| 完美时机 | ATB≥90% | ×1.15 |
| 仓促惩罚 | ATB<70% | ×0.8 |
| 动能加成 | 动能>0 | ×(1 + 动能%) |
| 元素克制 | 克制关系 | ×1.2 / ×0.9 |

### 7.2 DOT（持续伤害）设计

| 效果 | 基础伤害/秒 | 持续时间 | 可叠加 |
|------|------------|---------|--------|
| 灼烧 | 5 | 3秒 | 3层 |
| 感电 | 3 | 2秒 | 5层 |
| 霜冻 | 2 | 2秒 |减速叠加 |

### 7.3 控制效果设计

| 效果 | 持续时间 | 效果描述 |
|------|---------|---------|
| 冻结 | 2秒 | ATB暂停 |
| 眩晕 | 1秒 | 无法行动 |
| 减速 | 3秒 | ATB速度-30% |
| 定身 | 1秒 | ATB冻结0.5秒 |
| 致盲 | 2秒 | 50%未命中 |
